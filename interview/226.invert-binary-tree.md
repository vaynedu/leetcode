# 226. 翻转二叉树

## 题目
翻转二叉树（每个节点的左右子树交换）。

## 核心思想

**递归：交换左右子树**

- 终止条件：root == nil
- 递归翻转左右子树
- 交换当前节点的左右指针

## 代码实现

```go
func invertTree(root *TreeNode) *TreeNode {
    if root == nil {
        return nil
    }
    root.Left, root.Right = root.Right, root.Left
    invertTree(root.Left)
    invertTree(root.Right)
    return root
}
```

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(h) |
