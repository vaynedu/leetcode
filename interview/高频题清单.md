# 高频题清单

这个清单按面试复习优先级组织。训练顺序遵循：

```text
模式代表题 → 高频变体 → 易错边界 → 综合设计
```

每题复习时，不只看代码是否会写，还要检查四件事：

1. 能说清为什么属于这个模式。
2. 能说清核心不变量。
3. 能证明每一步为什么不漏答案。
4. 能指出最容易错的边界和错误后果。

## 每题复盘四问

每做完一题，在题解笔记里补这四句话：

1. 这题为什么属于这个模式？
2. 这个算法维护的不变量是什么？
3. 每一步为什么不会漏答案？
4. 哪个边界最容易错，错了会怎样？
