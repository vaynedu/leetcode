# 1644. 二叉树的最近公共祖先 II

## 核心思想

节点有 parent 指针，利用 parent 从两个节点向上遍历找 LCA。

## 实现

```go
func LowestCommonAncestorII(root, p, q *TreeNodeWithParent) *TreeNodeWithParent {
    // 收集 p 到根的路径
    pathP := map[*TreeNodeWithParent]bool{}
    for p != nil {
        pathP[p] = true
        p = p.Parent
    }
    // 从 q 向上找，第一个在 pathP 中的就是 LCA
    for q != nil {
        if pathP[q] { return q }
        q = q.Parent
    }
    return nil
}
```

## 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(h) |
| 空间 | O(h) |
