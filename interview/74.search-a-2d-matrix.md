# 74. 搜索二维矩阵

## 题目

编写一个高效的算法来判断 m x n 矩阵中是否存在一个目标值。该矩阵具有以下特性：
- 每行的元素从左到右升序排列
- 每列的第一个元素大于上一行的最后一个元素

## 难度

Medium

## 标签

数组 | 二分查找

## 示例

```
输入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3
输出：true

输入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 13
输出：false
```

## 核心思路

**方法：一次二分（坐标映射）**
- 将二维矩阵当作长度 m*n 的一维数组
- 下标 i 映射到 (i/n, i%n)
- 直接二分查找

**方法2：两次二分（先找行，再找列）**

## 关键点

- 映射公式：`row = mid / n`, `col = mid % n`
- 矩阵性质：每行首 > 上一行尾，等价于整体有序数组

## 复杂度

- 时间：O(log(m*n))
- 空间：O(1)
