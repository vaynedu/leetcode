# 704. 二分查找

## 一句话
**标准二分模板**：左闭右闭，while left <= right，mid 偏左

## 核心
```go
left, right := 0, len(nums)-1
for left <= right {
    mid := left + (right-left)/2
    if nums[mid] == target { return mid }
    else if nums[mid] < target { left = mid + 1 }
    else { right = mid - 1 }
}
return -1
```

## 记忆口诀
```
左闭右闭 <= 判断
mid 防溢要减左
大小比较定边界
找不到就返 -1
```

## 复杂度
- 时间: O(log N)
- 空间: O(1)
