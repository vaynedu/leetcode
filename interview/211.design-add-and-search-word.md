# 211. 添加与搜索单词 - 数据结构设计

## 核心思想

在 Trie 基础上增加 **'.' 通配符** 支持：'.' 可以匹配任意单个字符。

## 与 208 的区别

| 功能 | 208 | 211 |
|------|-----|-----|
| 插入 | ✓ | ✓ |
| 查找 | exact match | exact match + '.' |
| 前缀匹配 | ✓ | ✗ |

## Search 递归实现

```go
func (w *WordDictionary) searchHelper(word string, node *TrieNode211) bool {
    for i, ch := range word {
        if ch == '.' {
            // '.' → 遍历所有子节点，递归匹配剩余部分
            for _, child := range node.children {
                if w.searchHelper(word[i+1:], child) {
                    return true
                }
            }
            return false
        }
        if _, ok := node.children[ch]; !ok {
            return false
        }
        node = node.children[ch]
    }
    return node.isEnd
}
```

## 复杂度

| 操作 | 时间 | 空间 |
|------|------|------|
| AddWord | O(m) | O(m) |
| Search | O(m) 或 O(26^m) 含 '.' | O(m) |

## 关键点

1. **'.' 出现时**：遍历当前节点所有子节点，递归匹配剩余字符串
2. **'.' 没有子节点**：直接返回 false
3. **递归结束**：当 word 全部匹配完，检查 node.isEnd

## 模板口诀

```
Search 遇 '.' → 遍历 children，递归匹配剩余
Search 遇字母 → 精准匹配路径
结尾看 isEnd
```
