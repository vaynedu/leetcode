# 702. 搜索有序数组（未知长度）

## 题目
 SortedArray，Get(i) 返回第 i 个元素，超出范围返回 2^31-1。找目标索引。

## 核心思想

**双倍指针找上界 + 二分**

- 翻倍 right 找上界
- 在 [left, right] 内标准二分

## 代码实现

```go
func search(reader ArrayReader, target int) int {
    if reader.Get(0) == target {
        return 0
    }
    left, right := 0, 1
    for reader.Get(right) < target {
        left = right
        right *= 2
    }
    for left <= right {
        mid := left + (right-left)/2
        val := reader.Get(mid)
        if val == target {
            return mid
        } else if val < target {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return -1
}
```

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(log n) |
| 空间 | O(1) |
