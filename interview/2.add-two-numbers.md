# 2. 两数相加

## 题目
给定两个非空链表，表示两个非负整数，数字逆序存储，每位一个节点，返回和的链表。

## 核心思想

**模拟加法：逐位相加，记录进位**

- dummy 简化头节点处理
- 同时遍历两链表，叠加进位
- 循环终止：三链表都为空且无进位

## 代码实现

```go
func addTwoNumbers(l1 *ListNode, l2 *ListNode) *ListNode {
    dummy := &ListNode{}
    cur := dummy
    carry := 0
    for l1 != nil || l2 != nil || carry > 0 {
        sum := carry
        if l1 != nil {
            sum += l1.Val
            l1 = l1.Next
        }
        if l2 != nil {
            sum += l2.Val
            l2 = l2.Next
        }
        carry = sum / 10
        cur.Next = &ListNode{Val: sum % 10}
        cur = cur.Next
    }
    return dummy.Next
}
```

## trace — l1=[2,4,3], l2=[5,6,4]

| 位 | l1 | l2 | sum | digit | carry |
|----|----|----|-----|-------|-------|
| 个 | 2 | 5 | 7 | 7 | 0 |
| 十 | 4 | 6 | 10 | 0 | 1 |
| 百 | 3 | 4+1 | 8 | 8 | 0 |

答案：7→0→8

| 指标 | 值 |
|------|-----|
| 时间 | O(max(m,n)) |
| 空间 | O(max(m,n)) |
