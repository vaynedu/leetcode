# 322. 零钱兑换

## 题目
给定硬币面额数组 coins 和总金额 amount，求凑成 amount 的最少硬币数。

## 核心思想

**完全背包：dp[i] = 凑成金额 i 的最少硬币数**

- dp[0] = 0，其他初始化为 INF
- 对每种硬币，金额从小到大遍历（可重复使用）
- 状态转移：dp[i] = min(dp[i], dp[i-coin] + 1)

## 代码实现

```go
func coinChange(coins []int, amount int) int {
    dp := make([]int, amount+1)
    for i := range dp {
        dp[i] = math.MaxInt
    }
    dp[0] = 0
    for _, c := range coins {
        for i := c; i <= amount; i++ {
            if dp[i-c] != math.MaxInt {
                dp[i] = min(dp[i], dp[i-c]+1)
            }
        }
    }
    if dp[amount] == math.MaxInt {
        return -1
    }
    return dp[amount]
}
```

## trace — coins = [1, 2, 5], amount = 11

| i | dp[i] | 说明 |
|---|-------|------|
| 0 | 0 | 初始化 |
| 1 | 1 | dp[1] = min(inf, dp[0]+1) |
| 2 | 1 | dp[2] = min(inf, dp[1]+1) = 1 |
| 3 | 2 | dp[3] = min(inf, dp[2]+1) = 2 |
| 5 | 1 | dp[5] = min(inf, dp[4]+1) 但先到 5 |
| 11 | 3 | 5+5+1 = 3 |

答案：3（5+5+1）

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(amount × len(coins)) |
| 空间 | O(amount) |

## 关键点

- 金额从小到大遍历（完全背包）
- INF 初始化判断避免无效更新
- 如果要求输出具体方案，需要记录路径
