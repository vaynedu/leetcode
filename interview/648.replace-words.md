# 648. 单词替换

## 核心思想

在 Trie 中存储词根，从句子中查找每个单词的**最短词根**。

## 算法步骤

```
1. 将 dictionary 中所有词根插入 Trie
2. 遍历 sentence 的每个单词
3. 对每个单词，在 Trie 中逐字符查找：
   - 遇到节点 isEnd=true → 找到最短词根，返回
   - 路径中断 → 返回原单词
```

## 关键区别：最短词根

```go
func (t *Trie648) FindRoot(word string) string {
    node := t.root
    var rootBytes []byte
    for _, ch := range word {
        if _, ok := node.children[rune(ch)]; !ok {
            return word  // 无路径 → 返回原单词
        }
        rootBytes = append(rootBytes, byte(ch))
        node = node.children[rune(ch)]
        if node.isEnd {
            return string(rootBytes)  // 找到词根 → 立即返回
        }
    }
    return word
}
```

## Trace 演示

```
dictionary = ["cat","bat","rat"]
sentence   = "the cattle was rattled by the baby"

cattle: c→a→t-t-l-e  路径 c-a-t 遇到 isEnd=true → "cat"
was:    w-a-s  路径 w-a 不存在 → 返回 "was"
rattled: r-a-t 遇到 isEnd=true → "rat"
baby:   b-a-b-y  路径 b-a 不存在 → 返回 "baby"

输出: "the cat was rat by the baby"
```

## 复杂度

| 操作 | 时间 | 空间 |
|------|------|------|
| 构建 Trie | O(d × L) | O(d × L) |
| 替换句子 | O(s × L) | O(1) |

d = dictionary 词根数，L = 平均长度，s = 句子单词数

## 模板口诀

```
Trie 存词根，FindRoot 找最短
isEnd 找到即返回，路径中断返原词
```
