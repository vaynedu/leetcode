# 92. 反转链表 II

## 题目
反转从位置 left 到 right 的链表节点（1-indexed），返回头节点。

## 核心思想

**头插法反转区间**

1. prev 移动到 left-1 位置
2. cur 指向待反转节点
3. 每次将 cur.Next 剪切到 prev 之后（头插）
4. 重复 right-left 次

## 代码实现

```go
func reverseBetween(head *ListNode, left int, right int) *ListNode {
    dummy := &ListNode{}
    dummy.Next = head
    prev := dummy
    for i := 0; i < left-1; i++ {
        prev = prev.Next
    }
    cur := prev.Next
    for i := 0; i < right-left; i++ {
        next := cur.Next
        cur.Next = next.Next
        next.Next = prev.Next
        prev.Next = next
    }
    return dummy.Next
}
```

## trace — [1,2,3,4,5], left=2, right=4

| 步骤 | 操作 | 结果 |
|------|------|------|
| 初始 | | 1→2→3→4→5 |
| 头插3 | 3插到1后 | 1→3→2→4→5 |
| 头插4 | 4插到1后 | 1→4→3→2→5 |

答案：1→4→3→2→5

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(1) |
