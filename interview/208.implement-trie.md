# 208. 实现 Trie (前缀树)

## 核心思想

Trie（发音 "try"）是一种树形数据结构，用于高效存储和检索字符串集合。

## Trie 结构

```
          root
         / | \
        t  a  b
       /   |   \
      o    p    a
     (to) (p)  (at)
```

- **根节点**不含字符
- 每条边代表一个字符
- **isEnd** 标记是否为单词结尾

## 实现要点

### 节点定义

```go
type TrieNode struct {
    children map[rune]*TrieNode
    isEnd    bool
}
```

### 插入 Insert

```go
func (t *Trie) Insert(word string) {
    node := t.root
    for _, ch := range word {
        if _, ok := node.children[ch]; !ok {
            node.children[ch] = NewTrieNode()
        }
        node = node.children[ch]
    }
    node.isEnd = true
}
```

### 查找 Search

```go
func (t *Trie) Search(word string) bool {
    node := t.searchPrefix(word)
    return node != nil && node.isEnd
}
```

### 前缀匹配 StartsWith

```go
func (t *Trie) StartsWith(prefix string) bool {
    return t.searchPrefix(prefix) != nil
}
```

## 复杂度

| 操作 | 时间 | 空间 |
|------|------|------|
| Insert | O(m) | O(m) |
| Search | O(m) | O(1) |
| StartsWith | O(m) | O(1) |

m = 字符串长度

## 模板口诀

```
Trie 两件宝：children map 建路径，isEnd 标记词尾
Search 结尾看 isEnd，StartsWith 结尾看节点
```
