# 79. 单词搜索

## 题目
给定二维字符 board 和单词 word，判断单词是否存在于网格中（相邻格子水平/垂直连接）。

## 核心思想

**DFS 回溯**

- 从每个格子作为起点尝试匹配
- 每步检查边界、是否访问过、字符是否匹配
- 匹配成功则递归四个方向
- 回溯：恢复 visited 状态

## 代码实现

```go
func exist(board [][]byte, word string) bool {
    if len(board) == 0 || len(word) == 0 {
        return false
    }
    m, n := len(board), len(board[0])
    visited := make([][]bool, m)
    for i := range visited {
        visited[i] = make([]bool, n)
    }

    var dfs func(i, j, idx int) bool
    dfs = func(i, j, idx int) bool {
        if idx == len(word) {
            return true
        }
        if i < 0 || i >= m || j < 0 || j >= n {
            return false
        }
        if visited[i][j] || board[i][j] != word[idx] {
            return false
        }
        visited[i][j] = true
        defer func() { visited[i][j] = false }()
        return dfs(i+1, j, idx+1) || dfs(i-1, j, idx+1) ||
               dfs(i, j+1, idx+1) || dfs(i, j-1, idx+1)
    }

    for i := 0; i < m; i++ {
        for j := 0; j < n; j++ {
            if dfs(i, j, 0) {
                return true
            }
        }
    }
    return false
}
```

## trace — board = [["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]], word = "ABCCED"

| 步骤 | 说明 |
|------|------|
| A→B→C→C→E | 找到路径 |
| D在最后不匹配 | 继续找 |

答案：true

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(M×N×4^L) |
| 空间 | O(M×N) |

## 关键点

- visited 记录已访问路径
- 剪枝：board[i][j] != word[idx] 提前返回
