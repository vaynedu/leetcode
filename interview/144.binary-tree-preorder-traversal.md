# 144. 二叉树前序遍历

## 核心思想

前序遍历：根 → 左 → 右

## 递归实现

```go
func Preorder144(root *TreeNode) []int {
    var result []int
    preorderRec(root, &result)
    return result
}

func preorderRec(node *TreeNode, result *[]int) {
    if node == nil { return }
    *result = append(*result, node.Val)  // 根
    preorderRec(node.Left, result)        // 左
    preorderRec(node.Right, result)       // 右
}
```

## 迭代实现

```go
func Preorder144Iter(root *TreeNode) []int {
    stack := []*TreeNode{root}
    for len(stack) > 0 {
        node := stack[len(stack)-1]
        stack = stack[:len(stack)-1]
        result = append(result, node.Val)
        if node.Right != nil { stack = append(stack, node.Right) }
        if node.Left != nil { stack = append(stack, node.Left) }
    }
}
```

## 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(h) 递归栈深度 |
