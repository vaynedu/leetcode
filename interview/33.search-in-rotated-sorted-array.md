# 33. 搜索旋转排序数组

## 一句话
**旋转数组二分**：先判断哪半边有序，再决定 target 在哪半边

## 核心
```go
if nums[left] <= nums[mid] { // 左半边有序
    if nums[left] <= target && target < nums[mid] {
        right = mid - 1
    } else {
        left = mid + 1
    }
} else { // 右半边有序
    if nums[mid] < target && target <= nums[right] {
        left = mid + 1
    } else {
        right = mid - 1
    }
}
```

## 记忆口诀
```
旋转数组二分查
先判左半边是否有序
有序则看 target 在不在
在则缩右，不在则扩左
无序同理看右半边
```

## 复杂度
- 时间: O(log N)
- 空间: O(1)
