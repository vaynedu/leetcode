# 225. 用队列实现栈

## 题目
用队列实现栈的 push、pop、top、empty 操作。

## 核心思想

**单队列模拟栈**

- push 时：新元素加入队尾，再把前面元素依次移到后面
- 这样队首始终是最后加入的元素（栈顶）
- pop/top 直接操作队首

## 代码实现

```go
func (this *MyStack) Push(x int) {
    this.queue = append(this.queue, x)
    for i := 0; i < len(this.queue)-1; i++ {
        this.queue = append(this.queue, this.queue[0])
        this.queue = this.queue[1:]
    }
}
```

## 复杂度

| 操作 | 时间 |
|------|------|
| push | O(n) |
| pop | O(1) |
| top | O(1) |
| empty | O(1) |
