# 42. 接雨水

## 题目
给定 n 个非负整数表示宽度为 1 的柱形图，计算下雨后能接多少雨水。

## 核心思想

**双指针：左右指针从两端向中间**

- 每个位置能接的雨水 = min(左侧最高, 右侧最高) - 当前高度
- 双指针：从低的一侧计算（保证另一侧有更高墙）
- leftMax/rightMax 记录遍历过程中的最高墙

## 代码实现

```go
func trap(height []int) int {
    if len(height) == 0 {
        return 0
    }
    left, right := 0, len(height)-1
    leftMax, rightMax := 0, 0
    water := 0
    for left < right {
        if height[left] < height[right] {
            if height[left] >= leftMax {
                leftMax = height[left]
            } else {
                water += leftMax - height[left]
            }
            left++
        } else {
            if height[right] >= rightMax {
                rightMax = height[right]
            } else {
                water += rightMax - height[right]
            }
            right--
        }
    }
    return water
}
```

## trace — [0,1,0,2,1,0,1,3,2,1,2,1]

| 位置 | 高度 | 左max | 右max | 雨水 |
|------|------|-------|-------|------|
| 2 | 0 | 1 | 3 | 1 |
| 5 | 0 | 1 | 3 | 2 |
| 6 | 1 | 1 | 2 | 1 |
| 累计 | | | | 6 |

答案：6

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(1) |

## 关键点

- 从低侧计算：如果 height[left] < height[right]，则左边一定不会是短板
- DP 版本：先预处理 leftMax/rightMax 数组
- 单调栈解法：处理边界情况
