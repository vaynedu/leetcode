# 300. 最长递增子序列

## 题目
给定数组 nums，找最长递增子序列的长度（子序列不要求连续）。

## 核心思想

**二分 + tails 数组：tails[i] = 长度为 i+1 的递增子序列的最小尾部值**

- 遍历每个元素，用二分找到第一个 >= 当前值的位置
- 替换或追加 tails 数组
- tails 长度即为 LIS 长度

## 代码实现

```go
func lengthOfLIS(nums []int) int {
    tails := make([]int, 0)
    for _, v := range nums {
        i := 0
        for i < len(tails) && tails[i] < v {
            i++
        }
        if i == len(tails) {
            tails = append(tails, v)
        } else {
            tails[i] = v
        }
    }
    return len(tails)
}
```

## trace — [10, 9, 2, 5, 3, 7, 101, 18]

| v | tails 变化 | 说明 |
|---|-----------|------|
| 10 | [10] | 第一值 |
| 9 | [9] | 替换 10 |
| 2 | [2] | 替换 9 |
| 5 | [2, 5] | 追加 |
| 3 | [2, 3] | 替换 5 |
| 7 | [2, 3, 7] | 追加 |
| 101 | [2, 3, 7, 101] | 追加 |
| 18 | [2, 3, 7, 18] | 替换 101 |

答案：4，[2, 3, 7, 18]

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n log n) |
| 空间 | O(n) |

## 关键点

- DP 暴力是 O(n²)，二分优化到 O(n log n)
- 严格递增用 <，非严格递增用 <=（取决于题目）
