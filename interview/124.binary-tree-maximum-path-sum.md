# 124. 二叉树中的最大路径和

## 题目
找二叉树中任意节点到任意节点的路径最大和（路径至少含一个节点）。

## 核心思想

**后序遍历 + 动态规划**

- dfs 返回：经过该节点且向下延伸的最大贡献
- 路径可以是：左→根→右（拐点）、或只选一边、或只有根
- 更新全局最大：pathSum = root + max(0,left) + max(0,right)

## 代码实现

```go
func maxPathSum(root *TreeNode) int {
    maxSum := -1 << 63
    var dfs func(node *TreeNode) int
    dfs = func(node *TreeNode) int {
        if node == nil {
            return 0
        }
        left := dfs(node.Left)
        right := dfs(node.Right)
        pathSum := node.Val + max(0, left) + max(0, right)
        maxSum = max(maxSum, pathSum)
        return node.Val + max(0, max(left, right))
    }
    dfs(root)
    return maxSum
}
```

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(h) |
