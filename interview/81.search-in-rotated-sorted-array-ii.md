# 81. 搜索旋转排序数组 II

## 题目

已知一个长度为 n 的升序数组在某个位置被旋转了（元素个数可能为 0）。给定一个 target，如果 target 在数组中返回 true，否则返回 false。数组中可能包含重复元素。

## 难度

Medium

## 标签

数组 | 二分查找

## 示例

```
输入：nums = [2,5,6,0,0,1,2], target = 0
输出：true

输入：nums = [2,5,6,0,0,1,2], target = 3
输出：false
```

## 核心思路

与33题类似，但需处理重复导致**无法判断哪半有序**的情况：

**判断哪半有序：**
```
if nums[left] <= nums[mid]   → 左半边有序
else                         → 右半边有序
```

**关键去重逻辑：**
当 `nums[left] == nums[mid] == nums[right]` 时，两侧都无法确定有序，此时 `left++` 收缩边界。

## 关键点

- `nums[left] == nums[mid] == nums[right]` 时 left++ 跳过，无法确定 mid 在哪半
- 最坏情况 O(n)，例如 `[1,1,1,1,1,1,1]` 找 target = 0

## 复杂度

- 时间：O(n) 最坏
- 空间：O(1)
