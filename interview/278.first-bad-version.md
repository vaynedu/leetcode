# 278. 第一个错误的版本

## 一句话
**左边界模板**：找第一个 isBadVersion(mid) == true

## 核心
```go
left, right := 1, n
for left < right {
    mid := left + (right-left)/2
    if isBadVersion(mid) {
        right = mid
    } else {
        left = mid + 1
    }
}
return left
```

## 记忆口诀
```
版本从1开始计
bad在左就缩右
good在左就扩右
最终 left == right
```

## 复杂度
- 时间: O(log N)
- 空间: O(1)
