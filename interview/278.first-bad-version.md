# 278. 第一个错误的版本

## 题目
isBadVersion(mid) 判断 mid 是否为坏版本，找第一个坏版本索引。

## 核心思想

**二分查找**

- isBadVersion(mid) == true → 坏版本在左侧
- isBadVersion(mid) == false → 坏版本在右侧

## 代码实现

```go
func firstBadVersion(n int) int {
    left, right := 1, n
    for left < right {
        mid := left + (right-left)/2
        if isBadVersion(mid) {
            right = mid
        } else {
            left = mid + 1
        }
    }
    return left
}
```

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(log n) |
| 空间 | O(1) |
