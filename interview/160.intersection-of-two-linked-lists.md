# 160. 相交链表

## 题目
返回两个链表相交的节点，若不相交返回 nil。

## 核心思想

**双指针同步走**

- p1 从 A 出发，走完到 B
- p2 从 B 出发，走完到 A
- 若相交：p1 和 p2 走的总长度相等（a+b+c = b+a+c）
- 若不相交：同时走到 nil 返回 nil

## 代码实现

```go
func getIntersectionNode(headA, headB *ListNode) *ListNode {
    if headA == nil || headB == nil {
        return nil
    }
    p1, p2 := headA, headB
    for p1 != p2 {
        if p1 == nil {
            p1 = headB
        } else {
            p1 = p1.Next
        }
        if p2 == nil {
            p2 = headA
        } else {
            p2 = p2.Next
        }
    }
    return p1
}
```

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(m+n) |
| 空间 | O(1) |
