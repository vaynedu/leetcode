# 22. 括号生成

## 题目
给定 n 对括号，生成所有有效组合。

## 核心思想

**回溯 + 剪枝**

只有左括号数量严格大于右括号时，才能放右括号。

```
backtrack(left, right):
    if left == right == n:
        保存结果
        return
    if left < n:  加 '('
    if left > right: 加 ')'
```

## 代码实现

```go
func generateParenthesis(n int) []string {
    res := []string{}
    track := make([]byte, 0, 2*n)

    var backtrack func(left, right int)
    backtrack = func(left, right int) {
        if left == n && right == n {
            res = append(res, string(track))
            return
        }
        if left < n {
            track = append(track, '(')
            backtrack(left+1, right)
            track = track[:len(track)-1]
        }
        if right < left {
            track = append(track, ')')
            backtrack(left, right+1)
            track = track[:len(track)-1]
        }
    }

    backtrack(0, 0)
    return res
}
```

## trace — n=3，5个有效组合

```
(
├── (
│   ├── (
│   │   └── ) → ((()))
│   └── )
│       ├── ( → (()())
│       └── ) → (())()
└── )
    └── ( → ()()
        ├── ( → ()(())
        └── ) → ()()()
```

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(4^n / sqrt(n))，第 n 个卡特兰数 |
| 空间 | O(n)，递归深度 |

## 卡特兰数

有效括号数 = C(2n, n) / (n+1)，增长率 ~ 4^n / (n^1.5)
