# 70. 爬楼梯

## 题目
每次可以爬 1 或 2 个台阶，问爬到第 n 阶有多少种方法。

## 核心思想

**dp[i] = dp[i-1] + dp[i-2]**

到第 i 阶 = 到第 i-1 阶爬1步 + 到第 i-2 阶爬2步。

空间优化：只保留前两个状态（斐波那契）。

## 代码实现

```go
func climbStairs(n int) int {
    if n <= 2 {
        return n
    }
    prev2, prev1 := 1, 2
    for i := 3; i <= n; i++ {
        curr := prev1 + prev2
        prev2, prev1 = prev1, curr
    }
    return prev1
}
```

## trace — n=5

| i | dp[i] |
|---|-------|
| 1 | 1 (1) |
| 2 | 2 (1+1, 2) |
| 3 | 3 (1+1+1, 1+2, 2+1) |
| 4 | 5 |
| 5 | 8 |

答案：8

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(1) |
