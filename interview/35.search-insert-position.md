# 35. 搜索插入位置

## 一句话
**左边界模板**：左闭右开 [left, right)，找第一个 >= target

## 核心
```go
left, right := 0, len(nums)  // 右开
for left < right {           // <
    mid := left + (right-left)/2
    if nums[mid] < target {
        left = mid + 1
    } else {
        right = mid           // 不减1
    }
}
return left                   // 返回 left
```

## 记忆口诀
```
左闭右开不取等
nums[mid] < target 往右走
否则 right = mid
返回 left 第一个大于等于
```

## 复杂度
- 时间: O(log N)
- 空间: O(1)
