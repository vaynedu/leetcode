# 643. 子数组最大平均数（Maximum Average Subarray I）

## 题目信息
- 题号：LeetCode 643
- 标题：Maximum Average Subarray I
- 难度：简单（Easy）

---

## 一句话思路
**固定窗口**：先算第一个窗口和，右移时减左加右，O(1) 更新。

---

## 核心思路

### 固定窗口滑动
```
1. 计算第一个窗口 [0, k-1] 的和
2. 窗口右移：sum += nums[R] - nums[L]
3. 每次更新最大平均值
```

---

## 关键点/考点
```
时间：O(N)
空间：O(1)

追问：
- 为什么不用除以 k？→ 最后再除，节省计算
- 浮点数精度？→ 用 float64
```

---

## 易错点
```
1. 用 float64 计算平均值
2. 第一个窗口要单独处理
3. 除法用 float64(sum) / float64(k)
```

---

## 记忆口诀
```
固定窗口滑起来，
先算首窗求和。
右移减左加右，
O(1) 更新无碍。
```

---

## Go 代码实现

```go
func findMaxAverage(nums []int, k int) float64 {
    // 第一个窗口的和
    sum := 0
    for i := 0; i < k; i++ {
        sum += nums[i]
    }
    maxSum := sum

    // 滑动窗口
    for i := k; i < len(nums); i++ {
        sum += nums[i] - nums[i-k]
        if sum > maxSum {
            maxSum = sum
        }
    }

    return float64(maxSum) / float64(k)
}
```

---

## 测试用例

### 正常情况
- `nums=[1,12,-5,-6,50,3], k=4` → 12.75
- `nums=[5], k=1` → 5.0

### 边界情况
- `k=1` → 单个元素最大值
- `k=len(nums)` → 整个数组平均值

### 极端情况
- 全正数：`[1,2,3,4,5], k=3` → 4.0
- 全负数：`[-5,-4,-3,-2], k=2` → -3.5
