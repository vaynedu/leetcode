# 145. 二叉树后序遍历

## 核心思想

后序遍历：左 → 右 → 根

## 递归实现

```go
func Postorder145(root *TreeNode) []int {
    var result []int
    postorderRec(root, &result)
    return result
}

func postorderRec(node *TreeNode, result *[]int) {
    if node == nil { return }
    postorderRec(node.Left, result)        // 左
    postorderRec(node.Right, result)       // 右
    *result = append(*result, node.Val)   // 根
}
```

## 迭代实现（技巧）

```go
func Postorder145Iter(root *TreeNode) []int {
    stack := []*TreeNode{root}
    for len(stack) > 0 {
        node := stack[len(stack)-1]
        stack = stack[:len(stack)-1]
        result = append(result, node.Val) // 根先访问
        if node.Left != nil { stack = append(stack, node.Left) }
        if node.Right != nil { stack = append(stack, node.Right) }
    }
    // 根→右→左 → reverse → 左→右→根
    for i, j := 0, len(result)-1; i < j; i, j = i+1, j-1 {
        result[i], result[j] = result[j], result[i]
    }
}
```
