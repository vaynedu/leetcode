# 20. 有效括号（Valid Parentheses）

## 题目信息
- 题号：LeetCode 20
- 标题：Valid Parentheses
- 难度：简单（Easy）

---

## 一句话思路
**栈**：左括号进栈，右括号匹配出栈，最终栈空为有效。

---

## 核心思路

### 关键规则
- 左括号：`(`, `[`, `{` → 进栈
- 右括号：`)`, `]`, `}` → 匹配栈顶
- 有效条件：遍历完 + 栈为空

### 步骤
```
1. 初始化空栈
2. 遍历每个字符：
   - 左括号 → 进栈
   - 右括号 →
     a. 栈空 → return false
     b. 栈顶不匹配 → return false
     c. 匹配 → 出栈
3. 遍历完，栈为空 → return true
```

---

## 关键点/考点
```
时间：O(N)
空间：O(N) 栈

追问：
- 为什么用栈？→ 括号匹配具有最近相邻性，栈 FILO 特性天然对应
- 栈空说明什么？→ 所有左括号都有对应的右括号
- 如何处理不匹配？→ 任何一步不匹配直接返回 false
```

---

## 易错点
```
1. 遍历完栈可能不为空（如 "([]"）→ 缺少右括号
2. 右括号来时栈可能已空（如 "]()"）→ 缺少左括号
3. 栈顶类型要用 map 正确映射：')' → '('，']' → '['，'}' → '{'
```

---

## 记忆口诀
```
左括号进栈，右括号出栈。
栈空遇右括，直接返回假。
遍历完栈空，匹配成功啦。
```

---

## Go 代码实现

```go
func isValid(s string) bool {
    stack := []rune{}
    pairs := map[rune]rune{')': '(', ']': '[', '}': '{'}

    for _, c := range s {
        if c == '(' || c == '[' || c == '{' {
            stack = append(stack, c) // 左括号进栈
        } else {
            // 右括号：栈空或不匹配 → 无效
            if len(stack) == 0 || stack[len(stack)-1] != pairs[c] {
                return false
            }
            stack = stack[:len(stack)-1] // 出栈
        }
    }
    return len(stack) == 0 // 栈空说明全部匹配
}
```

---

## 测试用例

### 正常情况
- `"()"` → true
- `"()[]{}"` → true
- `"{[]}"` → true
- `"((()))"` → true

### 边界情况
- `""` → true（空字符串）
- `"("` → false（只有左括号）
- `")"` → false（只有右括号）

### 错误情况
- `"(]"` → false
- `"([)]"` → false
- `"{[}]"` → false
- `"((())"` → false（缺少右括号）

### 极端情况
- `"()" .repeat(10000)` → true
- `"({}[])"` → true（多层嵌套）
