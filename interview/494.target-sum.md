# 494. 目标和

## 题目
给定数组 nums，每个元素前可加 + 或 -，使运算结果等于 target，返回方案数。

## 核心思想

**转化为子集和问题**

记正数和为 P，负数和为 N：
- P - N = target
- P + N = sum(nums)
- => **P = (sum + target) / 2**

问题变成：从 nums 中选若干元素凑成 P，计数。

## 代码实现

```go
func findTargetSumWays(nums []int, target int) int {
    sum := 0
    for _, v := range nums {
        sum += v
    }
    if sum < 0 {
        sum = -sum
    }
    if target < 0 {
        target = -target
    }
    if target > sum || (sum+target)%2 != 0 {
        return 0
    }
    P := (sum + target) / 2
    dp := make([]int, P+1)
    dp[0] = 1
    for _, v := range nums {
        for i := P; i >= v; i-- {
            dp[i] += dp[i-v]
        }
    }
    return dp[P]
}
```

## trace — nums = [1, 1, 1, 1, 1], target = 3

| 步骤 | 说明 |
|------|------|
| sum=5, target=3 | |
| P=(5+3)/2=4 | 凑成 4 的方案数 |

dp 结果：dp[4] = 5

答案：5（5 个元素加 +/- 凑成 3 有 5 种方案）

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n × P) |
| 空间 | O(P) |

## 关键点

- 转化思想：+ / - 分配转化为子集和
- target 可为负，先取绝对值
- dp[i] 存方案数，初始化 dp[0]=1
