# 373. 查找和最小的K对数

## 题目
给定两个升序数组 nums1 和 nums2，返回和最小的 K 对数（每对包含 nums1[i] 和 nums2[j]）。

## 核心思想

**最小堆 + 已访问剪枝**

从 (0,0) 开始（最小和的候选），每次弹出最小的，把它的下方 (i+1,j) 和右方 (i,j+1) 加入堆。

用 visited 防止重复入堆。

## 代码实现

```go
func kSmallestPairs(nums1 []int, nums2 []int, k int) [][]int {
    if len(nums1) == 0 || len(nums2) == 0 || k == 0 {
        return nil
    }

    h := &minPairHeap{}
    heap.Init(h)
    visited := make(map[[2]int]bool)

    heap.Push(h, pair{0, 0, nums1[0] + nums2[0]})
    visited[[2]int{0, 0}] = true

    res := [][]int{}
    for h.Len() > 0 && len(res) < k {
        p := heap.Pop(h).(pair)
        res = append(res, []int{nums1[p.i], nums2[p.j]})

        if p.i+1 < len(nums1) && !visited[[2]int{p.i + 1, p.j}] {
            heap.Push(h, pair{p.i + 1, p.j, nums1[p.i+1] + nums2[p.j]})
            visited[[2]int{p.i + 1, p.j}] = true
        }
        if p.j+1 < len(nums2) && !visited[[2]int{p.i, p.j + 1}] {
            heap.Push(h, pair{p.i, p.j + 1, nums1[p.i] + nums2[p.j+1]})
            visited[[2]int{p.i, p.j + 1}] = true
        }
    }
    return res
}
```

## trace — nums1=[1,7,11], nums2=[2,4,6], k=3

| Step | Pop | Pair | Sum | Push |
|------|-----|------|-----|------|
| 1 | (0,0) | [1,2] | 3 | (1,0)->[7,2]=9, (0,1)->[1,4]=5 |
| 2 | (0,1) | [1,4] | 5 | (1,1)->[7,4]=11, (0,2)->[1,6]=7 |
| 3 | (0,2) | [1,6] | 7 | (1,2)->[7,6]=13 |

结果：[[1,2],[1,4],[1,6]]

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(k log k)，最多 k 次出堆，每次 O(log k) |
| 空间 | O(k)，堆和 visited |
