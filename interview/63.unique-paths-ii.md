# 63. 不同路径 II

## 题目
网格中有障碍物（1），机器人不能经过障碍，问有多少条路径。

## 核心思想

和62题一样，区别在于遇到障碍时：
- obstacleGrid[i][j] == 1 → dp[j] = 0（不可达）
- 处理第一行/列时：遇到障碍后后面全部为0

## 代码实现

```go
func uniquePathsWithObstacles(obstacleGrid [][]int) int {
    m := len(obstacleGrid)
    n := len(obstacleGrid[0])
    dp := make([]int, n)

    for j := 0; j < n; j++ {
        if obstacleGrid[0][j] == 1 {
            dp[j] = 0
        } else if j == 0 {
            dp[j] = 1
        } else {
            dp[j] = dp[j-1]
        }
    }

    for i := 1; i < m; i++ {
        for j := 0; j < n; j++ {
            if obstacleGrid[i][j] == 1 {
                dp[j] = 0
            } else if j > 0 {
                dp[j] += dp[j-1]
            }
        }
    }
    return dp[n-1]
}
```

## trace

```
obstacleGrid = [
  [0, 0, 0]
  [0, 1, 0]   ← 障碍在 (1,1)
  [0, 0, 0]
]

dp after i=0: [1, 1, 1]
dp after i=1: [1, 0, 1]  ← (1,1) 障碍，dp[1]=0
dp after i=2: [1, 1, 2]
```

答案：2（只能绕开障碍）
