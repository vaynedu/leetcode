# 210. 课程表 II

## 题目信息

- 题号：LeetCode 210
- 标题：Course Schedule II
- 难度：中等（Medium）
- 模式：图 / 拓扑排序

## 一句话思路

和 207 一样做拓扑排序，只是把每次出队的课程追加到结果中；如果结果长度不足，说明有环。

## 为什么想到这个模式

题目要求返回一种可行的上课顺序。课程依赖是有向边 `b -> a`，合法上课顺序就是这个有向图的拓扑序。

如果图中有环，就不存在任何拓扑序，返回空数组。

## 状态定义 / 数据结构

- `graph[b]`：学完 `b` 后可以学习的后续课程。
- `inDegree[a]`：课程 `a` 剩余前置依赖数量。
- `queue`：当前入度为 0 的课程。
- `result`：拓扑排序结果。

## 核心不变量

`result` 中的课程都已经满足其所有前置依赖。

队列中保存当前所有可学习课程；弹出课程后，释放它指向的后续课程依赖，入度归零则入队。

## 手推全过程

`numCourses = 4, prerequisites = [[1,0],[2,0],[3,1],[3,2]]`

| 步骤 | 队列 | 出队 | result |
|------|------|------|--------|
| 初始 | `[0]` | - | `[]` |
| 1 | `[0]` | 0 | `[0]` |
| 2 | `[1,2]` | 1 | `[0,1]` |
| 3 | `[2]` | 2 | `[0,1,2]` |
| 4 | `[3]` | 3 | `[0,1,2,3]` |

`[0,2,1,3]` 也合法，因为 1 和 2 没有相互依赖。

## Go 代码实现

```go
func findOrder(numCourses int, prerequisites [][]int) []int {
    graph := make([][]int, numCourses)
    inDegree := make([]int, numCourses)
    for _, p := range prerequisites {
        a, b := p[0], p[1]
        graph[b] = append(graph[b], a)
        inDegree[a]++
    }

    queue := []int{}
    for i := 0; i < numCourses; i++ {
        if inDegree[i] == 0 {
            queue = append(queue, i)
        }
    }

    result := []int{}
    for len(queue) > 0 {
        course := queue[0]
        queue = queue[1:]
        result = append(result, course)
        for _, next := range graph[course] {
            inDegree[next]--
            if inDegree[next] == 0 {
                queue = append(queue, next)
            }
        }
    }
    if len(result) != numCourses {
        return []int{}
    }
    return result
}
```

## 错误写法与反例

### 错误 1：不检查结果长度

有环时队列可能很早为空，但 `result` 只包含部分课程。必须 `len(result) == numCourses` 才说明所有课程都可完成。

### 错误 2：要求唯一顺序

拓扑序不一定唯一。测试时应验证顺序合法，而不是只比较某一个固定答案。

### 错误 3：边方向建反

`[a,b]` 表示先学 `b` 再学 `a`，边是 `b -> a`。建反会输出不合法顺序。

## 四问复盘

1. 这题为什么属于这个模式？
   因为课程依赖要求输出一个满足所有有向边先后关系的顺序，就是拓扑排序。

2. 这个算法维护的不变量是什么？
   队列中的课程入度为 0，加入 `result` 的课程都已满足前置依赖。

3. 每一步为什么不会漏答案？
   每处理一门课都会更新所有后继入度，任何依赖被清空的课程都会入队。

4. 哪个边界最容易错，错了会怎样？
   最容易错的是有环时返回部分结果。必须长度不足就返回空数组。
