# 120. 三角形最小路径和

## 题目
给定三角形数组，从顶部到底层，找路径和最小的一条路径（每步只能走到下一行相邻的数）。

## 核心思想

**从底部向上 DP**

从最后一行往上：dp[j] = min(dp[j], dp[j+1]) + triangle[i][j]

底部往上只需要一行数组，边界干净。

## 代码实现

```go
func minimumTotal(triangle [][]int) int {
    n := len(triangle)
    dp := make([]int, n+1) // 多一个元素，dp[j+1] 不越界

    for i := n - 1; i >= 0; i-- {
        for j := 0; j <= i; j++ {
            dp[j] = min(dp[j], dp[j+1]) + triangle[i][j]
        }
    }
    return dp[0]
}
```

## trace

```
triangle = [
  [2],
  [3,4],
  [6,5,7],
  [4,1,8,3]
]

从底部往上:

i=3: dp = [min(∞,4)+4, min(4,1)+1, min(1,8)+8, min(8,3)+3]
      = [4, 1, 8, 3]

i=2: dp = [min(4,1)+6, min(1,8)+5, min(8,3)+7]
      = [7, 6, 10]

i=1: dp = [min(7,6)+3, min(6,10)+4]
      = [9, 10]

i=0: dp = [min(9,10)+2] = [11]
```

答案：11，路径 2→3→5→1

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n²) |
| 空间 | O(n) |

## 从上往下 vs 从下往上

从上往下需要两行数组（避免覆盖），从下往上只需一行，且代码更简洁。
