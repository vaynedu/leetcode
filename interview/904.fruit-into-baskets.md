# 904. 水果成篮（Fruit Into Baskets）

## 题目信息
- 题号：LeetCode 904
- 标题：Fruit Into Baskets
- 难度：中等（Medium）

---

## 一句话思路
**变长窗口 + map**：窗口内最多两种水果，统计每种水果的数量。

---

## 核心思路

### 题目抽象
- 数组中的值代表水果种类（数字）
- 最多选两种水果
- 找最长子数组

### 步骤
```
1. window 记录窗口内每种水果的数量
2. R 扩展，加入新水果
3. 当水果种类 > 2 时，L 收缩直到种类 <= 2
4. 更新最大长度
```

---

## 关键点/考点
```
时间：O(N)
空间：O(1)，最多两种水果

追问：
- 为什么用 map？→ 统计每种水果数量
- 收缩条件？→ 水果种类 > 2
- 什么时候更新答案？→ 每次窗口调整后
```

---

## 易错点
```
1. 收缩时要更新 map 中的计数
2. 收缩直到种类 <= 2，不是直接减到 1
3. 水果数量归 0 时要从 map 中删除（或特殊处理）
```

---

## 记忆口诀
```
窗口内最多两水果，
map 统计数量好。
种类超二就收缩，
收缩到二刚刚好。
```

---

## Go 代码实现

```go
func totalFruit(fruits []int) int {
    window := make(map[int]int)
    L := 0
    maxLen := 0

    for R := 0; R < len(fruits); R++ {
        window[fruits[R]]++

        // 种类超过 2，收缩左边界
        for len(window) > 2 {
            window[fruits[L]]--
            if window[fruits[L]] == 0 {
                delete(window, fruits[L])
            }
            L++
        }

        if R-L+1 > maxLen {
            maxLen = R - L + 1
        }
    }

    return maxLen
}
```

## 测试用例

### 正常情况
- `fruits=[1,2,1]` → 3
- `fruits=[1,2,3,2,2]` → 4（[2,3,2,2]）
- `fruits=[3,3,3,1,2,1,1,2,3,3,4]` → 5（[1,2,1,1,2]）

### 边界情况
- `fruits=[1]` → 1
- `fruits=[1,2]` → 2
- `fruits=[]` → 0

### 极端情况
- 全部相同：`[1,1,1,1]` → 4
- 交替：`[1,2,1,2,1,2]` → 2
