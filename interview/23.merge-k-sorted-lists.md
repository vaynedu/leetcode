# 23. 合并 K 个升序链表

## 题目

给你一个链表数组，每个链表都已经按升序排列。请你将所有链表合并到一个升序链表中，返回合并后的链表。

```
输入：lists = [[1,4,5],[1,3,4],[2,6]]
输出：1->1->2->3->4->4->5->6
```

## 难度

**频率**：🔥🔥🔥🔥🔥（面试高频Top10）

## 解题思路

### 方法一：分治归并（推荐）

```
分治思想：
1. 将 K 个链表分成两半
2. 递归处理左右两半
3. 合并两个有序链表

时间：O(N log K)，N 为总节点数，K 为链表数
空间：O(log K) 递归栈
```

### 方法二：最小堆

```
1. 用最小堆维护所有链表的当前节点
2. 每次弹出最小节点，接到结果链表
3. 如果弹出的节点还有下一个，加入堆

时间：O(N log K)
空间：O(K)
```

## 代码实现

### 方法一：分治归并

```go
package interview

type ListNode struct {
    Val  int
    Next *ListNode
}

// mergeKLists 分治归并 - O(N log K)
func mergeKLists(lists []*ListNode) *ListNode {
    if len(lists) == 0 {
        return nil
    }
    return mergeSort(lists, 0, len(lists)-1)
}

func mergeSort(lists []*ListNode, left, right int) *ListNode {
    // 递归终止条件
    if left == right {
        return lists[left]
    }
    // 防止整数溢出
    mid := left + (right-left)/2
    // 递归左右两半
    l1 := mergeSort(lists, left, mid)
    l2 := mergeSort(lists, mid+1, right)
    // 合并两个有序链表
    return mergeTwoLists(l1, l2)
}

// mergeTwoLists 合并两个有序链表
func mergeTwoLists(l1, l2 *ListNode) *ListNode {
    dummy := &ListNode{}
    cur := dummy
    for l1 != nil && l2 != nil {
        if l1.Val < l2.Val {
            cur.Next = l1
            l1 = l1.Next
        } else {
            cur.Next = l2
            l2 = l2.Next
        }
        cur = cur.Next
    }
    if l1 != nil {
        cur.Next = l1
    } else {
        cur.Next = l2
    }
    return dummy.Next
}
```

### 方法二：最小堆

```go
import "container/heap"

type MinHeap []*ListNode

func (h MinHeap) Len() int            { return len(h) }
func (h MinHeap) Less(i, j int) bool  { return h[i].Val < h[j].Val }
func (h MinHeap) Swap(i, j int)       { h[i], h[j] = h[j], h[i] }
func (h *MinHeap) Push(x any)         { *h = append(*h, x.(*ListNode)) }
func (h *MinHeap) Pop() any {
    n := len(*h)
    node := (*h)[n-1]
    *h = (*h)[:n-1]
    return node
}

func mergeKListsByHeap(lists []*ListNode) *ListNode {
    h := new(MinHeap)
    // 初始化堆：每个链表的第一个节点
    for _, list := range lists {
        if list != nil {
            heap.Push(h, list)
        }
    }
    dummy := &ListNode{}
    cur := dummy
    for h.Len() > 0 {
        node := heap.Pop(h).(*ListNode)
        cur.Next = node
        cur = cur.Next
        if node.Next != nil {
            heap.Push(h, node.Next)
        }
    }
    return dummy.Next
}
```

## 面试要点

```
1. 分治归并是标准解法，必须会写
2. 递归终止条件：left == right
3. mid 计算用 left + (right-left)/2 防止溢出
4. 合并两个有序链表是基础模块，必须熟练
5. 进阶：最小堆解法也要掌握

常见追问：
- 时间复杂度怎么算？
  → 每个节点参与 log K 次合并 = O(N log K)
- 空间复杂度？
  → 递归栈 O(log K)，堆 O(K)
```

```go
func TestMergeKLists(t *testing.T) {
    // 构建测试链表 [1->4->5]
    l1 := &ListNode{Val: 1}
    l1.Next = &ListNode{Val: 4}
    l1.Next.Next = &ListNode{Val: 5}

    // 构建测试链表 [1->3->4]
    l2 := &ListNode{Val: 1}
    l2.Next = &ListNode{Val: 3}
    l2.Next.Next = &ListNode{Val: 4}

    // 构建测试链表 [2->6]
    l3 := &ListNode{Val: 2}
    l3.Next = &ListNode{Val: 6}

    lists := []*ListNode{l1, l2, l3}
    result := mergeKLists(lists)

    // 验证结果
    expected := []int{1, 1, 2, 3, 4, 4, 5, 6}
    for _, v := range expected {
        if result == nil || result.Val != v {
            t.Errorf("mergeKLists failed, got %v", result)
        }
        if result != nil {
            result = result.Next
        }
    }

    // 空数组测试
    if mergeKLists([]*ListNode{}) != nil {
        t.Errorf("empty lists should return nil")
    }

    // 全 nil 测试
    if mergeKLists([]*ListNode{nil, nil}) != nil {
        t.Errorf("all nil should return nil")
    }
}
```
