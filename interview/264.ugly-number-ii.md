# 264. 丑数 II

## 题目
第 n 个丑数（只含质因数 2、3、5 的数）。

## 核心思想

**DP + 三指针**

- dp[i] = 第 i 个丑数
- 三指针 p2/p3/p5 指向下一个乘以 2/3/5 的位置
- 取最小值为下一个丑数，对应指针后移
- 自动去重（如 6=2×3=3×2）

## 代码实现

```go
func nthUglyNumber(n int) int {
    dp := make([]int, n)
    dp[0] = 1
    p2, p3, p5 := 0, 0, 0
    for i := 1; i < n; i++ {
        v2, v3, v5 := dp[p2]*2, dp[p3]*3, dp[p5]*5
        dp[i] = min(v2, min(v3, v5))
        if dp[i] == v2 {
            p2++
        }
        if dp[i] == v3 {
            p3++
        }
        if dp[i] == v5 {
            p5++
        }
    }
    return dp[n-1]
}
```

## trace — n=10

| i | dp[i] | p2,p3,p5 |
|---|-------|----------|
| 0 | 1 | 0,0,0 |
| 1 | 2 | 1,0,0 |
| 2 | 3 | 1,1,0 |
| 3 | 4 | 2,1,0 |
| 4 | 5 | 2,1,1 |
| 5 | 6 | 3,2,1 |
| ... | | |

答案：第10个丑数 = 12
