# 207. 课程表

## 题目
numCourses 门课， prerequisites[i] = [a, b] 表示课程 a 需要先上 b，判断能否完成所有课程。

## 核心思想

**BFS 拓扑排序 + 入度统计**

- 构建有向图，统计每个节点的入度
- 入度为 0 的节点可以先学，加入队列
- BFS 移除节点，更新相邻节点入度
- 若最终访问节点数 == numCourses，则无环

## 代码实现

```go
func canFinish(numCourses int, prerequisites [][]int) bool {
    graph := make([][]int, numCourses)
    inDegree := make([]int, numCourses)
    for _, p := range prerequisites {
        a, b := p[0], p[1]
        graph[b] = append(graph[b], a)
        inDegree[a]++
    }

    queue := []int{}
    for i := 0; i < numCourses; i++ {
        if inDegree[i] == 0 {
            queue = append(queue, i)
        }
    }

    visited := 0
    for len(queue) > 0 {
        course := queue[0]
        queue = queue[1:]
        visited++
        for _, next := range graph[course] {
            inDegree[next]--
            if inDegree[next] == 0 {
                queue = append(queue, next)
            }
        }
    }
    return visited == numCourses
}
```

- 检测环：visited != numCourses 则有环
