# 198. 房屋盗贼

## 题目
给定数组 nums 表示每间房的钱，不能偷连续两间，求能偷的最大金额。

## 核心思想

**dp[i] = max(dp[i-1], dp[i-2] + nums[i])**

解释：要么不偷第 i 间（保留 dp[i-1]），要么偷第 i 间（加上 dp[i-2]）。

## 代码实现

```go
func rob(nums []int) int {
    if len(nums) == 0 {
        return 0
    }
    if len(nums) == 1 {
        return nums[0]
    }
    prev2, prev1 := nums[0], max(nums[0], nums[1])
    for i := 2; i < len(nums); i++ {
        cur := max(prev1, prev2+nums[i])
        prev2 = prev1
        prev1 = cur
    }
    return prev1
}
```

## trace — [2, 7, 9, 3, 1]

| i | nums[i] | dp[i-2] | dp[i-1] | dp[i] | 说明 |
|---|---------|---------|---------|-------|------|
| 0 | 2 | - | - | 2 | 第一间 |
| 1 | 7 | - | 2 | 7 | max(2, 7) = 7 |
| 2 | 9 | 2 | 7 | 11 | max(7, 2+9) |
| 3 | 3 | 7 | 11 | 11 | max(11, 7+3) |
| 4 | 1 | 11 | 11 | 12 | max(11, 11+1) |

答案：12，偷第 1、3、5 间（索引 1,2,4 → 7+9+1）

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(1) |

## 关键点

- 不能连续偷 = 隔一个偷
- 状态只和前两个有关，滚动数组优化
- 变种：环形房屋（213）
