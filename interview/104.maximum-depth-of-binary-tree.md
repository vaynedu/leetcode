# 104. 二叉树最大深度

## 题目
给定二叉树根节点，返回最大深度（根到最远叶子的节点数）。

## 核心思想

**递归：depth = max(left, right) + 1**

- 终止条件：root == nil
- 递归计算左右子树深度
- 返回较大者 + 1

## 代码实现

```go
func maxDepth(root *TreeNode) int {
    if root == nil {
        return 0
    }
    left := maxDepth(root.Left)
    right := maxDepth(root.Right)
    if left > right {
        return left + 1
    }
    return right + 1
}
```

## trace — [3,9,20,null,null,15,7]

答案：3（根→右→右叶子）

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(h) |

## 关键点

- 递归终止条件容易漏
- BFS 层序也可求：计算层数
- 变种：最小深度（111）
