# 15. 三数之和

## 题目
给定数组 nums，找出所有三元组 [a,b,c] 使得 a+b+c=0，不含重复三元组。

## 核心思想

**排序 + 双指针**

- 排序后固定一个数 nums[i]，双指针找两数之和等于 -nums[i]
- nums[i] > 0 时后续不可能和为 0
- 去重：跳过与前一个相同的值

## 代码实现

```go
func threeSum(nums []int) [][]int {
    sort.Ints(nums)
    result := [][]int{}
    n := len(nums)
    for i := 0; i < n-2 && nums[i] <= 0; i++ {
        if i > 0 && nums[i] == nums[i-1] {
            continue
        }
        left, right := i+1, n-1
        for left < right {
            sum := nums[i] + nums[left] + nums[right]
            if sum == 0 {
                result = append(result, []int{nums[i], nums[left], nums[right]})
                left++
                right--
                for left < right && nums[left] == nums[left-1] {
                    left++
                }
                for left < right && nums[right] == nums[right+1] {
                    right--
                }
            } else if sum < 0 {
                left++
            } else {
                right--
            }
        }
    }
    return result
}
```

## trace — [-1,0,1,2,-1,-4]

排序后：[-4,-1,-1,0,1,2]

| i | nums[i] | left | right | sum | 结果 |
|---|---------|------|-------|-----|------|
| 0 | -4 | 1 | 5 | - | nums[i]>0 后停止 |

答案：[[-1,-1,2],[-1,0,1]]

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n²) |
| 空间 | O(1) |

## 关键点

- 排序是前提，双指针利用有序性
- 去重逻辑要仔细（i、left、right 三个都要去重）
- 变种：最接近的三数之和（16）、三数之和加一（923）
