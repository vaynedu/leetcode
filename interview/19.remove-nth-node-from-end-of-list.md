# 19. 删除链表的倒数第 N 个节点

## 题目
给定链表头节点，删除倒数第 n 个节点。

## 核心思想

**快慢指针**

- dummy 节点简化删除头节点的情况
- fast 先走 n+1 步
- 然后 fast、slow 同步走，直到 fast 到达末尾
- 此时 slow.Next 指向倒数第 n 个节点

## 代码实现

```go
func removeNthFromEnd(head *ListNode, n int) *ListNode {
    dummy := &ListNode{}
    dummy.Next = head
    fast := dummy
    slow := dummy
    for i := 0; i <= n; i++ {
        if fast == nil {
            return head
        }
        fast = fast.Next
    }
    for fast != nil {
        slow = slow.Next
        fast = fast.Next
    }
    slow.Next = slow.Next.Next
    return dummy.Next
}
```

## trace — head=[1,2,3,4,5], n=2

| 步骤 | fast | slow |
|------|------|------|
| 初始 | dummy | dummy |
| 先走3步 | 3 | dummy |
| 同步1 | 4 | 1 |
| 同步2 | 5 | 2 |
| 同步3 | nil | 3 |

slow.Next 即节点 4，删除后：1→2→3→5

答案：1→2→3→5

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(L) |
| 空间 | O(1) |

## 关键点

- n+1 步确保 slow 停在待删除节点的前一个
- dummy 统一处理头节点删除
- 两次遍历：第一次计数可换为一次（数学法）
