# 46. 全排列

## 题目
给定一个不含重复数字的数组，返回所有可能的排列。

## 核心思想

**回溯 + used 标记**

全排列和组合的区别：
- 组合：选/不选，下一层从 start 开始
- 排列：所有位置轮流放值，用 used 标记已使用

## 代码实现

```go
func permute(nums []int) [][]int {
    res := [][]int{}
    path := []int{}
    used := make([]bool, len(nums))

    var backtrack func()
    backtrack = func() {
        if len(path) == len(nums) {
            tmp := make([]int, len(path))
            copy(tmp, path)
            res = append(res, tmp)
            return
        }
        for i := 0; i < len(nums); i++ {
            if used[i] {
                continue
            }
            used[i] = true
            path = append(path, nums[i])
            backtrack()
            path = path[:len(path)-1]
            used[i] = false
        }
    }

    backtrack()
    return res
}
```

## trace — nums=[1,2,3]

```
[1] → [1,2] → [1,2,3] → 保存
         ↓回溯
      [1,3] → [1,3,2] → 保存
   ↓回溯
[2] → ...
[3] → ...
```

共 3! = 6 种排列。

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n × n!)，n! 个排列 |
| 空间 | O(n)，used 数组 |
