# 93. 复原 IP 地址

## 题目
给定只含数字的字符串，恢复所有可能的 IP 地址。

## 核心思想

**回溯：分成 4 段，每段 1-3 位**

- 递归尝试每段长度（1-3）
- 每段必须是 0-255，且无前导零（除非是 "0"）
- 递归到第 4 段时检查剩余字符串是否为空
- 用 "." 连接各段

## 代码实现

```go
func restoreIpAddresses(s string) []string {
    result := []string{}
    if len(s) < 4 || len(s) > 12 {
        return result
    }
    var backtrack func(s string, parts int, path []string)
    backtrack = func(s string, parts int, path []string) {
        if parts == 4 {
            if len(s) == 0 {
                result = append(result, join(path))
            }
            return
        }
        for i := 1; i <= 3 && i <= len(s); i++ {
            seg := s[:i]
            if validSegment(seg) {
                backtrack(s[i:], parts+1, append(path, seg))
            }
        }
    }
    backtrack(s, 0, []string{})
    return result
}
```
