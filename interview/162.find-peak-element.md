# 162. 寻找峰值

## 题目
峰值元素大于相邻元素，返回任意峰值索引。数组边界视为负无穷。

## 核心思想

**二分查找**

- 若 nums[mid] < nums[mid+1]，峰值在右侧（右侧必有峰值）
- 否则峰值在左侧（含 mid）

## 代码实现

```go
func findPeakElement(nums []int) int {
    left, right := 0, len(nums)-1
    for left < right {
        mid := left + (right-left)/2
        if nums[mid] < nums[mid+1] {
            left = mid + 1
        } else {
            right = mid
        }
    }
    return left
}
```

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(log n) |
| 空间 | O(1) |
