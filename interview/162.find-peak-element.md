# 162. 寻找峰值

## 一句话
**峰值二分查找**：比较 mid 和 mid+1，nums[mid] < nums[mid+1] 说明峰值在右半边

## 核心
```go
left, right := 0, len(nums)-1
for left < right {
    mid := left + (right-left)/2
    if nums[mid] < nums[mid+1] {
        left = mid + 1
    } else {
        right = mid
    }
}
return left
```

## 记忆口诀
```
峰值比邻都大
比较 mid 和 mid+1
小则往右走
大则缩右边
两端负无穷
必有峰值返
```

## 复杂度
- 时间: O(log N)
- 空间: O(1)
