# 141. 环形链表

## 题目
判断链表中是否有环。

## 核心思想

**快慢指针（Floyd 判圈）**

- 慢指针每次走一步，快指针每次走两步
- 若有环，快慢必在某点相遇
- 若无环，快指针先到达末尾

## 代码实现

```go
func hasCycle(head *ListNode) bool {
    if head == nil {
        return false
    }
    slow := head
    fast := head
    for fast.Next != nil && fast.Next.Next != nil {
        slow = slow.Next
        fast = fast.Next.Next
        if slow == fast {
            return true
        }
    }
    return false
}
```

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(1) |
