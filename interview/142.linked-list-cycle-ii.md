# 142. 环形链表 II

## 题目
返回环形链表的环入口节点，若无环返回 nil。

## 核心思想

**快慢指针 + 数学推导**

- 第一步：快慢指针相遇（必定在环内相遇）
- 第二步：设入口前有 a 步，环长 b
- 数学证明：2(a+x) = a + x + kb => a = (k-1)b + (b-x)
- 从头同步走 a 步，必与慢指针在入口相遇

## 代码实现

```go
func detectCycle(head *ListNode) *ListNode {
    if head == nil {
        return nil
    }
    slow := head
    fast := head
    for fast.Next != nil && fast.Next.Next != nil {
        slow = slow.Next
        fast = fast.Next.Next
        if slow == fast {
            ptr := head
            for ptr != slow {
                ptr = ptr.Next
                slow = slow.Next
            }
            return ptr
        }
    }
    return nil
}
```

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(1) |
