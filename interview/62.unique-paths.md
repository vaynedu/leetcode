# 62. 不同路径

## 题目
机器人从 m×n 网格左上角到右下角，只能向右或向下走，问有多少条不同路径。

## 核心思想

**dp[i][j] = dp[i-1][j] + dp[i][j-1]**

路径数 = 从上方来的路径 + 从左边来的路径。

空间优化：一行数组，从左到右更新。

## 代码实现

```go
func uniquePaths(m int, n int) int {
    dp := make([]int, n)
    for j := 0; j < n; j++ {
        dp[j] = 1
    }
    for i := 1; i < m; i++ {
        for j := 1; j < n; j++ {
            dp[j] += dp[j-1]
        }
    }
    return dp[n-1]
}
```

## trace — m=3, n=7

```
     j=0  j=1  j=2  j=3  j=4  j=5  j=6
i=0 [  1    1    1    1    1    1    1 ]
i=1 [  1    2    3    4    5    6    7 ]
i=2 [  1    3    6   10   15   21   28 ]
```

答案：28

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(m×n) |
| 空间 | O(n) |

## 数学解法

组合数：C(m+n-2, m-1)，从 (m+n-2) 步中选 (m-1) 步向下。
