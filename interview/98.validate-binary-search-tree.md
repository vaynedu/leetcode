# 98. 验证二叉搜索树

## 题目
判断给定二叉树是否为有效的 BST。

## 核心思想

**中序遍历 + 前驱比较**

- BST 中序遍历得到升序序列
- 遍历时记录前驱节点值，若当前节点 <= 前驱则 false
- 左子树 < 根 < 右子树的递归约束

## 代码实现

```go
func isValidBST(root *TreeNode) bool {
    var prev *int
    var inorder func(node *TreeNode) bool
    inorder = func(node *TreeNode) bool {
        if node == nil {
            return true
        }
        if !inorder(node.Left) {
            return false
        }
        if prev != nil && *prev >= node.Val {
            return false
        }
        prev = &node.Val
        return inorder(node.Right)
    }
    return inorder(root)
}
```

## trace — [2,1,3]

| 遍历 | prev | node.Val | 比较 | 结果 |
|------|------|----------|------|------|
| 左2 | nil | - | - | - |
| 根1 | 2 | 1 | 2>=1 | false |

答案：false（注意题目是 [2,1,3] 时根是 2）

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(h) |

## 关键点

- 不能只比较左右子节点，要传递上下界
- 中序遍历 O(n) 最常用
- 递归版本需传 min/max 边界
