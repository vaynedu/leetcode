# 617. 合并二叉树

## 核心思想

两棵树对应节点值相加，返回合并后的树（可原地）。

## 实现

```go
func MergeTrees(root1, root2 *TreeNode) *TreeNode {
    if root1 == nil { return root2 }
    if root2 == nil { return root1 }
    root1.Val += root2.Val
    root1.Left = MergeTrees(root1.Left, root2.Left)
    root1.Right = MergeTrees(root1.Right, root2.Right)
    return root1
}
```

## 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(min(m,n)) |
| 空间 | O(min(h1,h2)) |
