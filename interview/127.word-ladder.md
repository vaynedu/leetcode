# 127. 单词阶梯

## 题目
给定 beginWord、endWord 和单词列表，每次只改变一个字符，问从 beginWord 到 endWord 的最短转换序列长度。

## 核心思想

**BFS 最短路**

- 每次只改变一个字符，变换后的词必须在 wordSet 中
- BFS 逐层扩展，同层第一次到达就是最短
- 访问过的词从 wordSet 中移除（防止重复）

## 代码实现

```go
func ladderLength(beginWord string, endWord string, wordList []string) int {
    wordSet := map[string]bool{}
    for _, w := range wordList {
        wordSet[w] = true
    }
    if !wordSet[endWord] {
        return 0
    }
    delete(wordSet, beginWord)

    queue := []string{beginWord}
    step := 1
    for len(queue) > 0 {
        size := len(queue)
        for i := 0; i < size; i++ {
            word := queue[0]
            queue = queue[1:]
            chars := []byte(word)
            for j := 0; j < len(chars); j++ {
                original := chars[j]
                for c := 'a'; c <= 'z'; c++ {
                    if byte(c) == original {
                        continue
                    }
                    chars[j] = byte(c)
                    newWord := string(chars)
                    if newWord == endWord {
                        return step + 1
                    }
                    if wordSet[newWord] {
                        delete(wordSet, newWord)
                        queue = append(queue, newWord)
                    }
                }
                chars[j] = original
            }
        }
        step++
    }
    return 0
}
```

## trace — begin="hit", end="cog", list=["hot","dot","dog","lot","log","cog"]

| step | 队列 |
|------|------|
| 1 | hit |
| 2 | hot |
| 3 | dot, lot |
| 4 | dog, log |
| 5 | cog (找到) |

答案：5

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(N × L²) N词数 L词长 |
| 空间 | O(N × L) |

## 关键点

- BFS 保证找到最短路径
- wordSet 移除已访问词
- 双向 BFS 可优化：beginSet/endSet 从两端搜索
