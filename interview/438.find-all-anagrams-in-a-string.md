# 438. 找到所有字母异位词（Find All Anagrams in a String）

## 题目信息
- 题号：LeetCode 438
- 标题：Find All Anagrams in a String
- 难度：中等（Medium）

---

## 一句话思路
**变长窗口（固定长度）**：和 567 类似，但找到后要**记录所有起始位置**，不是立即返回。

---

## 核心思路

### 与 567 的区别
- 567：找到一个就返回 true
- 438：**记录所有满足条件的起始位置**

### 步骤
```
1. 统计 t 的字符计数 need
2. 窗口遍历 s，长度 = len(t)
3. valid == len(need) 时，记录 L 到结果
4. 继续滑动，不返回
```

---

## 关键点/考点
```
时间：O(N)
空间：O(1)

追问：
- 返回什么？→ 起始位置数组
- 为什么返回后还要继续？→ 可能有多于一个答案
- 和 567 区别？→ 567 找排列，438 找所有字母异位词
```

---

## 易错点
```
1. 记录的是 L（窗口左边界），不是 R
2. 窗口长度等于 len(t) 时才检查
3. valid++ 在 == need[c] 时，valid-- 在 == need[d] 时
```

---

## 记忆口诀
```
固定长度窗口滑，
need window 双映射。
等于时来 valid 加，
等于 len 记录 L。
找到不返回继续滑，
所有位置都记录。
```

---

## Go 代码实现

```go
func findAnagrams(s string, p string) []int {
    need := make(map[rune]int)
    window := make(map[rune]int)

    for _, c := range p {
        need[c]++
    }

    L, R := 0, 0
    valid := 0
    result := []int{}

    for R < len(s) {
        c := rune(s[R])
        if _, ok := need[c]; ok {
            window[c]++
            if window[c] == need[c] {
                valid++
            }
        }
        R++

        // 窗口长度等于 p 时，检查并收缩
        if R-L == len(p) {
            if valid == len(need) {
                result = append(result, L)
            }
            d := rune(s[L])
            if _, ok := need[d]; ok {
                if window[d] == need[d] {
                    valid--
                }
                window[d]--
            }
            L++
        }
    }

    return result
}
```

---

## 测试用例

### 正常情况
- `s="cbaebabacd", p="abc"` → `[0,6]`（"cba" 和 "bac"）
- `s="abab", p="ab"` → `[0,1,2]`

### 边界情况
- `s=""` → `[]`
- `p=""` → `[]`
- `len(p) > len(s)` → `[]`

### 极端情况
- `s="aaaaaaaaaaaaaa", p="a"` → `[0,1,2,...13]`
- `s="abcdef", p="z"` → `[]`
