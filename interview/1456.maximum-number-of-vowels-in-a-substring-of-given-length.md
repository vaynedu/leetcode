# 1456. 定长子串中元音的最大数目（Maximum Number of Vowels in a Substring of Given Length）

## 题目信息
- 题号：LeetCode 1456
- 标题：Maximum Number of Vowels in a Substring of Given Length
- 难度：中等（Medium）

---

## 一句话思路
**固定窗口**：元音判断用 set，滑动统计窗口内元音数量。

---

## 核心思路

### 固定窗口 + 元音计数
```
1. 用 set 或 map 快速判断元音
2. 先统计第一个窗口的元音数
3. 窗口右移：根据新字符和旧字符更新计数
4. 记录最大元音数
```

---

## 关键点/考点
```
时间：O(N)
空间：O(1)

追问：
- 元音：a, e, i, o, u（大写不算）
- 为什么不用每次遍历窗口？→ 固定窗口只需 O(1) 更新
```

---

## 易错点
```
1. 只计算小写元音，大写不算
2. 窗口右移时：减去离开的字符，加新增的字符
```

---

## 记忆口诀
```
元音集合心中装，
首窗统计开个好。
右移减左加右新，
最大值来替换。
```

## Go 代码实现

```go
func maxVowels(s string, k int) int {
    vowel := map[rune]bool{
        'a': true, 'e': true, 'i': true, 'o': true, 'u': true,
    }

    // 第一个窗口
    count := 0
    for i := 0; i < k && i < len(s); i++ {
        if vowel[rune(s[i])] {
            count++
        }
    }
    maxCount := count

    // 滑动窗口
    for i := k; i < len(s); i++ {
        if vowel[rune(s[i])] {
            count++
        }
        if vowel[rune(s[i-k])] {
            count--
        }
        if count > maxCount {
            maxCount = count
        }
    }

    return maxCount
}
```

---

## 测试用例

### 正常情况
- `s="abciiidef", k=3` → 2
- `s="aeiou", k=2` → 2

### 边界情况
- `k=1` → 单字符，最多 1
- `k=len(s)` → 整个字符串

### 极端情况
- 全元音：`"aeiouaeiou", k=5` → 5
- 无元音：`"bcdfg", k=3` → 0
- 混合：`"python", k=2` → 0
