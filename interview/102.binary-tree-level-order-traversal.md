# 102. 二叉树层序遍历

## 题目

给你二叉树的根节点 root，返回其层序遍历（即逐层地从左到右访问所有节点）。

```
输入：root = [3,9,20,null,null,15,7]
     3
    / \
   9  20
     /  \
    15   7

输出：[[3], [9,20], [15,7]]
```

## 难度

**频率**：🔥🔥🔥🔥🔥（面试高频 BFS 模板题）

## 解题思路

### 核心数据结构

```
队列（Queue）：先进先出，保证按层访问
```

### 关键点

```
1. 用队列存储待访问的节点
2. 每一轮开始前，记录当前队列长度（即当前层的节点数）
3. 只循环取出这 n 个节点（取出时把它们的子节点加入队列，不影响当前层计数）
4. n 个取完后，一层遍历完，继续下一轮
```

### 图解

```
初始队列：[3]
输出：[[]]

第1轮：
  queue.len() = 1，取 1 个节点
  取出 3，把 9、20 入队
  输出：[3]

第2轮：
  queue.len() = 2，取 2 个节点
  取出 9（无子节点），取出 20，把 15、7 入队
  输出：[3], [9,20]

第3轮：
  queue.len() = 2，取 2 个节点
  取出 15、7（均无子节点）
  输出：[3], [9,20], [15,7]

队列为空，结束
```

## 代码实现

```go
package interview

type TreeNode struct {
	Val   int
	Left  *TreeNode
	Right *TreeNode
}

// levelOrder 二叉树层序遍历 - BFS + 队列
func levelOrder(root *TreeNode) [][]int {
	if root == nil {
		return nil
	}

	result := [][]int{}
	queue := []*TreeNode{root}

	for len(queue) > 0 {
		// 1. 记录当前层节点数
		size := len(queue)

		// 2. 遍历当前层的所有节点
		level := []int{}
		for i := 0; i < size; i++ {
			// 出队
			node := queue[0]
			queue = queue[1:]
			level = append(level, node.Val)

			// 左右子节点入队
			if node.Left != nil {
				queue = append(queue, node.Left)
			}
			if node.Right != nil {
				queue = append(queue, node.Right)
			}
		}

		// 3. 当前层结果加入
		result = append(result, level)
	}

	return result
}
```

## 面试要点

```
1. 为什么用队列而不是栈？
   → 队列先进先出，按层访问；栈后进先出，是深度优先

2. 如何区分每一层？
   → 用当前队列长度（size）控制循环次数

3. 时间复杂度？
   → O(N)，每个节点访问一次

4. 空间复杂度？
   → O(N)，队列最多存一层节点，最坏 O(N)
```

## 测试

```go
func TestLevelOrder(t *testing.T) {
	fmt.Println("=============================================")
	fmt.Println("       102. 二叉树层序遍历")
	fmt.Println("=============================================")
	fmt.Println()

	// 构建测试树
	//     3
	//    / \
	//   9  20
	//     /  \
	//    15   7
	root := &TreeNode{Val: 3}
	root.Left = &TreeNode{Val: 9}
	root.Right = &TreeNode{Val: 20}
	root.Right.Left = &TreeNode{Val: 15}
	root.Right.Right = &TreeNode{Val: 7}

	result := levelOrder(root)
	expected := [][]int{{3}, {9, 20}, {15, 7}}

	if !sliceEqual(result, expected) {
		t.Errorf("levelOrder failed")
	}

	// 空树测试
	if levelOrder(nil) != nil {
		t.Errorf("nil tree should return nil")
	}

	// 单节点测试
	single := &TreeNode{Val: 1}
	if r := levelOrder(single); len(r) != 1 || r[0][0] != 1 {
		t.Errorf("single node failed")
	}

	fmt.Println("✅ levelOrder - BFS + 队列 - PASS")
}

func sliceEqual(a, b [][]int) bool {
	if len(a) != len(b) {
		return false
	}
	for i := range a {
		if len(a[i]) != len(b[i]) {
			return false
		}
		for j := range a[i] {
			if a[i][j] != b[i][j] {
				return false
			}
		}
	}
	return true
}
```
