# 230. 二叉搜索树中第 K 小的元素

## 题目
返回 BST 中第 k 小的元素（1-indexed）。

## 核心思想

**中序遍历**

- BST 中序遍历得到升序序列
- 第 k 次访问即为第 k 小的元素
- 找到后提前返回

## 代码实现

```go
func kthSmallest(root *TreeNode, k int) int {
    count := 0
    result := 0
    var inorder func(node *TreeNode)
    inorder = func(node *TreeNode) {
        if node == nil {
            return
        }
        inorder(node.Left)
        count++
        if count == k {
            result = node.Val
            return
        }
        inorder(node.Right)
    }
    inorder(root)
    return result
}
```

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(H + k) |
| 空间 | O(H) |
