# 24. 两两交换链表中的节点

## 题目
每两个节点交换一次，返回交换后的链表头节点。

## 核心思想

**三指针迭代**

- dummy 简化头交换
- prev 指向已处理部分尾部
- 每轮：first=prev.Next, second=first.Next
- 交换 first 和 second，更新指针
- prev 移动到 first（交换后的第二个位置）

## 代码实现

```go
func swapPairs(head *ListNode) *ListNode {
    dummy := &ListNode{}
    dummy.Next = head
    prev := dummy
    for prev.Next != nil && prev.Next.Next != nil {
        first := prev.Next
        second := first.Next
        first.Next = second.Next
        second.Next = first
        prev.Next = second
        prev = first
    }
    return dummy.Next
}
```

## trace — [1,2,3,4]

| 步骤 | before | after |
|------|--------|-------|
| 1 | 1↔2 | 2→1→3→4 |
| 2 | 3↔4 | 2→1→4→3 |

答案：2→1→4→3

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(1) |
