# 114. 二叉树展开为链表

## 题目
将二叉树原地展开为"前序遍历"的单向链表。

## 核心思想

**后序遍历递归**

- 先递归处理左右子树
- 将左子树展开后接到根的右指针
- 左子树置空
- 找到原右子树的最右节点，连接原右子树

## 代码实现

```go
func flatten(root *TreeNode) {
    if root == nil {
        return
    }
    flatten(root.Left)
    flatten(root.Right)
    if root.Left == nil {
        return
    }
    right := root.Right
    root.Right = root.Left
    root.Left = nil
    cur := root.Right
    for cur.Right != nil {
        cur = cur.Right
    }
    cur.Right = right
}
```

展开后：1→2→3→4→5→6

|------|-----|
| 时间 | O(n) |
