# 84. 柱状图中最大的矩形

## 题目
给定 n 个非负整数表示柱形图宽度为 1，求最大矩形面积。

## 核心思想

**单调递增栈**

- 维护递增高度的下标
- 遇到比栈顶小的高度时，弹出栈顶计算以该高度为高的矩形
- 宽度 = 当前索引 - 栈下一项 - 1
- 末尾加哨兵 0 弹出所有剩余柱形

## 代码实现

```go
func largestRectangleArea(heights []int) int {
    heights = append(heights, 0)
    stack := []int{}
    maxArea := 0
    for i, h := range heights {
        for len(stack) > 0 && heights[stack[len(stack)-1]] >= h {
            height := heights[stack[len(stack)-1]]
            stack = stack[:len(stack)-1]
            width := i
            if len(stack) > 0 {
                width = i - stack[len(stack)-1] - 1
            }
            maxArea = max(maxArea, height*width)
        }
        stack = append(stack, i)
    }
    return maxArea
}
```

## trace — [2,1,5,6,2,3]

| i | h | 弹出 | 面积 | maxArea |
|---|----|------|------|--------|
| 1 | 1 | 2 | 2×1=2 | 2 |
| 5 | 0 | 6,5,1 | 5×1=5,6×2=12 | 12 |

答案：10（5×2）

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(n) |

## 关键点

- 单调栈核心：保持栈内高度递增
- 哨兵技巧：末尾加 0 确保全部弹出
- 变种：最大矩形（85）
