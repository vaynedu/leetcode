# 105. 从前序与中序遍历序列构造二叉树

## 题目
给定前序遍历 preorder 和中序遍历 inorder，重建二叉树。

## 核心思想

**递归：前序首为根，中序定位左右**

- 前序遍历：根→左→右，首个元素必为根
- 在中序遍历中定位根，左侧为左子树，右侧为右子树
- 递归构建左右子树

## 代码实现

```go
func buildTree(preorder []int, inorder []int) *TreeNode {
    if len(preorder) == 0 {
        return nil
    }
    root := &TreeNode{Val: preorder[0]}
    idx := 0
    for idx < len(inorder) && inorder[idx] != preorder[0] {
        idx++
    }
    root.Left = buildTree(preorder[1:idx+1], inorder[:idx])
    root.Right = buildTree(preorder[idx+1:], inorder[idx+1:])
    return root
}
```

## trace — pre=[3,9,20,15,7], in=[9,3,15,20,7]

| 步 | 根 | 左子树 | 右子树 |
|----|----|--------|--------|
| 根 | 3 | pre=[9], in=[9] | pre=[20,15,7], in=[15,20,7] |
| 右 | 20 | pre=[15], in=[15] | pre=[7], in=[7] |

答案：3←9, 20←15,7
