# 437. 路径总和 III

## 核心思想

路径起点可以是任意节点（不只是根），方向必须向下。

## 前缀和解法

```go
func PathSumIII(root *TreeNode, target int) int {
    prefix := map[int]int{0: 1}
    return helper(root, 0, target, prefix)
}

func helper(node *TreeNode, currSum, target int, prefix map[int]int) int {
    if node == nil { return 0 }
    currSum += node.Val
    count := prefix[currSum-target]   // 找补数
    prefix[currSum]++
    count += helper(node.Left, currSum, target, prefix)
    count += helper(node.Right, currSum, target, prefix)
    prefix[currSum]--                  // 回溯
    return count
}
```

## 核心公式

```
当前前缀和 = currSum
需要: currSum - target 在 map 中出现过
count = map[currSum - target]
```

## 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(n) |
| 空间 | O(h) |
