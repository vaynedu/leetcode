# 56. 合并区间

## 题目

给出一个区间的集合，合并所有重叠的区间。

```
输入: [[1,3],[2,6],[8,10],[15,18]]
输出: [[1,6],[8,10],[15,18]]
解释: 区间 [1,3] 和 [2,6] 重叠，合并为 [1,6]。

输入: [[1,4],[4,5]]
输出: [[1,5]]
解释: 区间 [1,4] 和 [4,5] 可被视为重叠的。
```

## 难度

**频率**：🔥🔥🔥🔥（面试高频）

## 解题思路

### 核心思想

```
1. 按区间的起始位置排序
2. 遍历区间，当前区间与上一个区间比较：
   - 如果当前起始 > 上一个结束：不重叠，加入结果
   - 如果当前起始 <= 上一个结束：重叠，取 max(上结束,  cur结束)
```

### 图解

```
排序前：[[1,3],[2,6],[8,10],[15,18]]
按起始排序：[[1,3],[2,6],[8,10],[15,18]]
           ↑
           1 < 3，重叠 → 合并为 [1,6]

遍历过程：
[1,3] -> 结果
[2,6] -> 2 <= 3，重叠，合并为 [1,6]
[8,10] -> 8 > 6，不重叠，加入
[15,18] -> 15 > 10，不重叠，加入

结果：[[1,6],[8,10],[15,18]]
```

## 代码实现

```go
package interview

// Interval 区间
type Interval struct {
	Start int
	End   int
}

// merge 合并区间
func merge(intervals []Interval) []Interval {
	if len(intervals) == 0 {
		return nil
	}

	// 1. 按起始位置排序
	sortByStart(intervals)

	// 2. 遍历合并
	result := []Interval{intervals[0]}

	for i := 1; i < len(intervals); i++ {
		last := &result[len(result)-1]
		curr := intervals[i]

		// 重叠：当前起始 <= 上一个结束
		if curr.Start <= last.End {
			// 合并，取较大的结束位置
			if curr.End > last.End {
				last.End = curr.End
			}
		} else {
			// 不重叠，直接加入
			result = append(result, curr)
		}
	}
	return result
}

// sortByStart 按起始位置排序
func sortByStart(intervals []Interval) {
	for i := 0; i < len(intervals)-1; i++ {
		for j := i + 1; j < len(intervals); j++ {
			if intervals[i].Start > intervals[j].Start {
				intervals[i], intervals[j] = intervals[j], intervals[i]
			}
		}
	}
}
```

## 面试要点

```
1. 为什么先排序？
   → 排序后重叠的区间必然相邻，便于合并

2. 排序的时间复杂度？
   → O(N log N)

3. 合并的时间复杂度？
   → O(N)

4. 边界情况：
   - intervals 为空
   - 只有一个区间
   - 所有区间都不重叠
   - 所有区间都重叠

5. 原地排序 vs 新建结果数组
```

func TestMergeIntervals(t *testing.T) {
	fmt.Println("=============================================")

			[]Interval{{1, 6}, {8, 10}, {15, 18}},
		},
		{
			[]Interval{{1, 4}, {4, 5}},
			[]Interval{{1, 5}},
		},
		{
			[]Interval{{1, 4}},
			[]Interval{{1, 4}},
		},
		{
			[]Interval{},
			nil,
		},
	}

	for i, tt := range tests {
		result := merge(tt.input)
		if ! intervalsEqual(result, tt.expected) {
			t.Errorf("Test %d failed", i+1)
		}
	}

	fmt.Println("✅ merge - 排序 + 贪心合并 - PASS")
}

func intervalsEqual(a, b []Interval) bool {
	if len(a) != len(b) {
		return false
	}
	for i := range a {
		if a[i].Start != b[i].Start || a[i].End != b[i].End {
			return false
		}
	}
	return true
}
