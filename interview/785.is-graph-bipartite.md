# 785. 判断二分图

## 题目
判断给定的无向图是否为二分图（节点可以分为两组，图的每条边连接两个不同组的节点）。

## 核心思想

**BFS 染色**

- 选一个未染色节点，染成颜色 1
- BFS 遍历，相邻节点染相反颜色
- 若发现相邻节点颜色相同，则不是二分图
- 多个连通分量需要逐个检查

## 代码实现

```go
func isBipartite(graph [][]int) bool {
    n := len(graph)
    color := make([]int, n)
    for i := 0; i < n; i++ {
        if color[i] != 0 {
            continue
        }
        color[i] = 1
        queue := []int{i}
        for len(queue) > 0 {
            node := queue[0]
            queue = queue[1:]
            for _, neighbor := range graph[node] {
                if color[neighbor] == 0 {
                    color[neighbor] = -color[node]
                    queue = append(queue, neighbor)
                } else if color[neighbor] == color[node] {
                    return false
                }
            }
        }
    }
    return true
}
```

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(V+E) |
| 空间 | O(V) |
