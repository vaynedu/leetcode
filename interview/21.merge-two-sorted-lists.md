# 21. 合并两个有序链表

## 题目
合并两个升序链表为一个升序链表。

## 核心思想

**迭代 + 虚拟头节点**

- dummy 节点简化边界处理
- 比较 l1、l2 当前节点，小的接入结果链表
- 任一链表走完后，直接连接剩余部分

## 代码实现

```go
func mergeTwoLists(l1 *ListNode, l2 *ListNode) *ListNode {
    dummy := &ListNode{}
    cur := dummy
    for l1 != nil && l2 != nil {
        if l1.Val <= l2.Val {
            cur.Next = l1
            l1 = l1.Next
        } else {
            cur.Next = l2
            l2 = l2.Next
        }
        cur = cur.Next
    }
    if l1 != nil {
        cur.Next = l1
    }
    if l2 != nil {
        cur.Next = l2
    }
    return dummy.Next
}
```

## trace — l1=[1,2,4], l2=[1,3,4]

| 步骤 | l1 | l2 | result |
|------|----|----|--------|
| 1 | 1 | 1 | 1 |
| 2 | 2 | 1 | 1→1 |
| 3 | 2 | 3 | 1→1→2 |
| 4 | 4 | 3 | 1→1→2→3 |
| 5 | 4 | 4 | 1→1→2→3→4 |
| 结束 | - | 4 | 1→1→2→3→4→4 |

答案：1→1→2→3→4→4

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(m+n) |
| 空间 | O(1) |

## 关键点

- dummy 节点避免处理头节点为空的情况
- 递归版本也简洁但空间 O(m+n)
