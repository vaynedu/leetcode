# 96. 不同的二叉搜索树

## 题目
给定整数 n（1≤n≤n），计算能组成多少棵不同的 BST（值 1 到 n）。

## 核心思想

**卡特兰数**

G[n] = sum(G[i-1] × G[n-i])，i 为根节点

G[0] = 1（空树），G[1] = 1

## 代码实现

```go
func numTrees(n int) int {
    if n <= 1 {
        return 1
    }
    dp := make([]int, n+1)
    dp[0] = 1
    dp[1] = 1

    for i := 2; i <= n; i++ {
        for j := 1; j <= i; j++ {
            dp[i] += dp[j-1] * dp[i-j]
        }
    }
    return dp[n]
}
```

## trace — n=3

dp[0] = 1, dp[1] = 1  
dp[2] = dp[0]×dp[1] + dp[1]×dp[0] = 1 + 1 = 2  
dp[3] = dp[0]×dp[2] + dp[1]×dp[1] + dp[2]×dp[0] = 2 + 1 + 2 = 5

| n | 卡特兰数 |
|---|---------|
| 0 | 1 |
| 1 | 1 |
| 2 | 2 |
| 3 | 5 |
| 4 | 14 |
| 5 | 42 |

答案：5

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n²) |
| 空间 | O(n) |
