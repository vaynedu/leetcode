# 295. 数据流中位数

## 题目
设计一个支持两种操作的数据结构：
- `addNum(num)` — 插入一个整数
- `findMedian()` — 返回当前所有数的中位数

中位数：奇数个时是排序后中间那个，偶数个时是中间两个的平均值。

## 核心思想

**两个堆：大顶堆 + 小顶堆**

```
大顶堆（maxH）：存较小的一半，堆顶 = 小的一半中最大的
小顶堆（minH）：存较大的一半，堆顶 = 大的一半中最小的

始终保持：len(maxH) >= len(minH)
```

中位数：
- 奇数：maxH 堆顶
- 偶数：(maxH 堆顶 + minH 堆顶) / 2

## 代码实现

```go
func (m *medianFinder) AddNum(num int) {
    heap.Push(m.maxH, num)
    heap.Push(m.minH, heap.Pop(m.maxH)) // 平衡：把 maxH 堆顶移到 minH
    if m.maxH.Len() < m.minH.Len() {
        // maxH 少了，拉回一个
        heap.Push(m.maxH, heap.Pop(m.minH))
    }
}

func (m *medianFinder) FindMedian() float64 {
    if m.maxH.Len() > m.minH.Len() {
        return float64((*m.maxH)[0])
    }
    return float64((*m.maxH)[0]+(*m.minH)[0]) / 2.0
}
```
