# 53. 最大子数组和

## 题目
给定整数数组，找连续子数组的最大和（至少包含一个元素）。

## 核心思想

**Kadane 算法：dp[i] = max(dp[i-1] + nums[i], nums[i])**

解释：要么把当前元素接在前面的子数组上，要么从当前元素重新开始（前面的和是负数拖累）。

## 代码实现

```go
func maxSubArray(nums []int) int {
    maxSum := nums[0]
    prev := nums[0]
    for i := 1; i < len(nums); i++ {
        prev = max(prev+nums[i], nums[i])
        maxSum = max(maxSum, prev)
    }
    return maxSum
}
```

## trace — [-2, 1, -3, 4, -1, 2, 1, -5, 4]

| i | nums[i] | dp[i] | maxSum |
|---|---------|-------|--------|
| 0 | -2 | -2 | -2 |
| 1 | 1 | max(-2+1, 1) = 1 | 1 |
| 2 | -3 | max(1-3, -3) = -2 | 1 |
| 3 | 4 | max(-2+4, 4) = 4 | 4 |
| 4 | -1 | max(4-1, -1) = 3 | 4 |
| 5 | 2 | max(3+2, 2) = 5 | 5 |
| 6 | 1 | max(5+1, 1) = 6 | 6 |
| 7 | -5 | max(6-5, -5) = 1 | 6 |
| 8 | 4 | max(1+4, 4) = 5 | 6 |

答案：6，子数组 [4,-1,2,1]

| 指标 | 值 |
|------|-----|
| 时间 | O(n) |
| 空间 | O(1) |
