# 416. 分割等和子集

## 题目
给定数组 nums，判断能否分成两个元素和相等的子集。

## 核心思想

**子集背包：转化为是否存在子集和为 sum/2**

- 总和为奇数直接 false
- 问题变成：能否从 nums 中选若干元素凑成 target
- dp[i] = 是否能凑成金额 i（01 背包，倒序遍历）

## 代码实现

```go
func canPartition(nums []int) bool {
    sum := 0
    for _, v := range nums {
        sum += v
    }
    if sum%2 != 0 {
        return false
    }
    target := sum / 2
    dp := make([]bool, target+1)
    dp[0] = true
    for _, v := range nums {
        for i := target; i >= v; i-- {
            dp[i] = dp[i] || dp[i-v]
        }
    }
    return dp[target]
}
```

## trace — [1, 5, 11, 5]

| sum | target | 说明 |
|-----|--------|------|
| 22 | 11 | 总和 22 |

dp 过程：
- v=1: dp[1]=T, dp[2..11] 逐步更新
- v=5: dp[5]=T, dp[6]=T, ...
- v=11: dp[11]=T（5+5+1）
- v=5: dp[11] 保持 T

答案：true

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(n × sum/2) |
| 空间 | O(sum/2) |

## 关键点

- 01 背包：金额从大到小遍历（保证每个元素只用一次）
- 降维优化：dp 数组用 bool
- 变种：目标和（494）、组合总和（377）
