# 90. 子集 II

## 题目
给定含重复数字的数组，返回所有不重复的子集。

## 核心思想

**回溯 + 排序去重（同层剪枝）**

和 78 的区别：数组有重复，同层遇到相同值只选第一个分支。

```go
if i > start && nums[i] == nums[i-1] {
    continue
}
```

注意是 `i > start`（同层）不是 `i > 0`（全局）。

## 代码实现

```go
func subsetsWithDup(nums []int) [][]int {
    sort.Ints(nums)
    res := [][]int{}
    path := []int{}

    var backtrack func(start int)
    backtrack = func(start int) {
        tmp := make([]int, len(path))
        copy(tmp, path)
        res = append(res, tmp)

        for i := start; i < len(nums); i++ {
            if i > start && nums[i] == nums[i-1] {
                continue
            }
            path = append(path, nums[i])
            backtrack(i + 1)
            path = path[:len(path)-1]
        }
    }

    backtrack(0)
    return res
}
```

## trace — nums=[1,2,2]

排序后 [1,2,2]

```
[]                    ← 每个节点都是子集
├── [1]
│   ├── [1,2]         ← 选第一个2
│   │   └── [1,2,2]   ← 选第二个2
│   └── [1,2]         ← 选第二个2（跳过）
└── [2]               ← 选第一个2（跳过第二个2的分支）
    └── [2,2]         ← 选第二个2

结果: [], [1], [1,2], [1,2,2], [2], [2,2] → 6 个
```

## 与 78 的区别

| | 78 | 90 |
|--|----|----|
| nums | 无重复 | 有重复 |
| 剪枝 | 无 | 同层相同值跳过 |
| 结果数 | 2^n | ≤ 2^n |
