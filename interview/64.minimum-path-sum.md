# 64. 最小路径和

## 题目
给定 m×n 网格，每个格子有非负权重，从左上角到右下角路径的最小和（只能向右或向下）。

## 核心思想

**dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]**

空间优化：原地修改 grid（第一行/列特殊处理）。

## 代码实现

```go
func minPathSum(grid [][]int) int {
    m, n := len(grid), len(grid[0])

    for j := 1; j < n; j++ {
        grid[0][j] += grid[0][j-1]
    }
    for i := 1; i < m; i++ {
        grid[i][0] += grid[i-1][0]
    }

    for i := 1; i < m; i++ {
        for j := 1; j < n; j++ {
            grid[i][j] += min(grid[i-1][j], grid[i][j-1])
        }
    }
    return grid[m-1][n-1]
}
```

## trace

```
grid = [
  [1, 3, 1]
  [1, 5, 1]
  [4, 2, 1]
]

Step 1 (第一行):   [1, 4, 5]
Step 2 (第一列):   [1, 2, 6]
Step 3 (其余):
  (1,1): 2 + min(4,5) = 2+4 = 6
  (1,2): 5 + min(5,6) = 5+5 = 10
  (2,1): 4 + min(2,10) = 4+2 = 6
  (2,2): 1 + min(6,10) = 1+6 = 7
```

答案：7，路径 1→3→1→1→1
