# 34. 在排序数组中查找元素的第一个和最后一个位置

## 题目

给定一个按照升序排列的整数数组 `nums`，和一个目标值 `target`。找出给定目标值在数组中开始和结束的位置。如果数组中不存在目标值，返回 `[-1, -1]`。

## 难度

Medium

## 标签

数组 | 二分查找

## 示例

```
输入：nums = [5,7,7,8,8,10], target = 8
输出：[3, 4]

输入：nums = [5,7,7,8,8,10], target = 6
输出：[-1, -1]

输入：nums = [], target = 0
输出：[-1, -1]
```

## 核心思路

**两次二分**：
1. **找左侧边界**：`lowerBound` 返回第一个 `>= target` 的位置
   - 若越界或该位置值 != target，说明不存在
2. **找右侧边界**：对 `target+1` 做一次 `lowerBound`，再减 1

两次都是标准的**左闭右开**二分模板。

## 关键点

- 两次二分，第二次是对 `target+1` 做 lowerBound
- `lowerBound` 使用 `[left, right)` 区间，`left == right` 时结束
- 边界条件：`left == len(nums)` 时表示所有元素都小于 target

## 复杂度

- 时间：O(log n)（两次二分）
- 空间：O(1)
