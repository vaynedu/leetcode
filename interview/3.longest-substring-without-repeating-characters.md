# 3. 无重复最长子串（Longest Substring Without Repeating Characters）

## 题目信息
- 题号：LeetCode 3
- 标题：Longest Substring Without Repeating Characters
- 难度：中等（Medium）

---

## 一句话思路
**滑动窗口 + HashMap**：右指针扩展，遇重复字符左指针收缩，HashMap 记录字符最近位置。

---

## 核心思路

### 暴力解（会超时）
枚举所有子串，检查每个子串是否有重复 → O(N³) 或 O(N²)

### 滑动窗口优化（O(N)）
- 窗口 [L, R] 表示当前无重复子串
- R 右移扩展，map 记录字符最近出现位置
- 遇重复字符：L 收缩到重复字符的下一个位置
- 全程只遍历一次

### 关键公式
```
当字符 c 在窗口内重复时：
L = max(L, lastPos[c] + 1)

maxLen = max(maxLen, R - L + 1)
```

---

## 关键点/考点
```
时间：O(N)，每个字符最多访问两次（一次R，一次L）
空间：O(min(N, 字符集大小))

追问：
- 为什么 L 用 max？→ 防止 L 被拉回到窗口外
- 为什么不删 map 只移动 L？→ 只移动 L，重复字符的位置已经是旧的了
- 字符集大小？→ ASCII 128 或 Unicode
```

---

## 易错点
```
1. L 只能往右走，不能往左 → 必须用 max(L, pos+1)
2. pos >= L 判断是否在窗口内，不是 pos >= 0
3. 先更新 maxLen，再更新 last 位置
4. last map 的值是位置，不是次数
```

---

## 记忆口诀
```
滑动窗口左右走，
右扩左缩记心头。
重复字符位置跳，
L 只进不退用 max。
每步更新最大长，
无重复子串就做完。
```

---

## Go 代码实现

```go
func lengthOfLongestSubstring(s string) int {
    last := make(map[rune]int) // 记录字符上次出现的位置
    L := 0
    maxLen := 0

    for R, c := range s {
        // 如果字符在窗口内重复，L 收缩到重复字符之后
        if pos, exists := last[c]; exists && pos >= L {
            L = pos + 1
        }
        last[c] = R      // 更新字符位置
        if R-L+1 > maxLen {
            maxLen = R - L + 1
        }
    }
    return maxLen
}
```

---

## 测试用例

### 正常情况
- `"abcabcbb"` → 3（"abc"）
- `"bbbbb"` → 1（"b"）
- `"pwwkew"` → 3（"wke"）
- `"abba"` → 2（"ab" 或 "ba"）

### 边界情况
- `""` → 0（空字符串）
- `" "` → 1（单个空格）
- `"a"` → 1

### 极端情况
- 全部不同："abcdefghijklmnopqrstuvwxyz" → 26
- 全部相同："aaaaaaaaaa" → 1
- 混合："abcdeffedcba" → 6（"edcba" 或逆序）
