# 69. x 的平方根

## 一句话
**二分求平方根**：mid <= x/mid 防止溢出，向下取整

## 核心
```go
if x < 2 { return x }
left, right := 1, x/2
for left <= right {
    mid := left + (right-left)/2
    if mid <= x/mid {
        left = mid + 1
    } else {
        right = mid - 1
    }
}
return right
```

## 记忆口诀
```
x小于2直接返
right 设为 x/2
mid 用除法防溢出
最终返回 right
```

## 复杂度
- 时间: O(log N)
- 空间: O(1)
