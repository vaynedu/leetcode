# 212. 单词搜索 II

## 核心思想

**Trie + DFS 回溯**：将所有单词存入 Trie，从网格每个格子出发 DFS 匹配 Trie 路径。

## 算法步骤

```
1. 将 words 所有单词插入 Trie
2. 遍历 board 每个格子 (i,j)
3. 从 (i,j) 出发 DFS：
   - 在 Trie 中找 board[i][j] 对应的子节点
   - 如果是 isEnd 节点 → 收集单词（去重）
   - 标记已访问 ('#')，四个方向递归
   - 恢复访问标记
```

## 去重技巧

找到单词后，将 `isEnd = false`，防止从其他路径重复找到。

## 代码实现

```go
func FindWords(board [][]byte, words []string) []string {
    trie := NewTrie212()
    for _, w := range words {
        trie.Insert(w)
    }
    m, n := len(board), len(board[0])
    var result []string

    var dfs func(i, j int, node *TrieNode212)
    dfs = func(i, j int, node *TrieNode212) {
        ch := rune(board[i][j])
        next, ok := node.children[ch]
        if !ok {
            return
        }

        if next.isEnd {
            result = append(result, next.word)
            next.isEnd = false // 去重
        }

        board[i][j] = '#'
        dirs := [4][2]int{{-1,0},{1,0},{0,-1},{0,1}}
        for _, d := range dirs {
            ni, nj := i+d[0], j+d[1]
            if ni >= 0 && ni < m && nj >= 0 && nj < n && board[ni][nj] != '#' {
                dfs(ni, nj, next)
            }
        }
        board[i][j] = byte(ch)
    }

    for i := 0; i < m; i++ {
        for j := 0; j < n; j++ {
            dfs(i, j, trie.root)
        }
    }
    return result
}
```

## 复杂度

| 指标 | 复杂度 |
|------|--------|
| 时间 | O(m × n × 4^L) L=最大单词长度 |
| 空间 | O(w × L) w=单词数 |

## 关键点

```
Trie 存所有单词 → 提供高效前缀查询
DFS 探索网格 → 匹配 Trie 路径
isEnd 去重 → 防止重复收集
'#' 标记 → 防止走回头路
```

## 模板口诀

```
Trie 存词表，DFS 走网格
isEnd 收集并去重，'#' 防止回头
四个方向扩展，Trie 引导方向
```
