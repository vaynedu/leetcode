# 367. 有效的完全平方数

## 一句话
**平方根变形**：用 x/mid 判断，最后验证 left*left == num

## 核心
```go
if num < 2 { return true }
left, right := 1, num/2
for left <= right {
    mid := left + (right-left)/2
    if mid <= num/mid {
        left = mid + 1
    } else {
        right = mid - 1
    }
}
return left*left == num || right*right == num
```

## 记忆口诀
```
小于2直接返
除法防溢来判断
最后验证两候选
left和right谁平方等于
```

## 复杂度
- 时间: O(log N)
- 空间: O(1)
