# 367. 有效的完全平方数

## 题目
判断一个数是否为完全平方数。

## 核心思想

**二分查找（避免乘法溢出）**

- mid == num/mid 且 num%mid == 0 判断完全平方
- mid > num/mid → 搜索左半边

## 代码实现

```go
func isPerfectSquare(num int) bool {
    if num < 2 {
        return true
    }
    left, right := 2, num/2
    for left <= right {
        mid := left + (right-left)/2
        if mid == num/mid && num%mid == 0 {
            return true
        } else if mid > num/mid {
            right = mid - 1
        } else {
            left = mid + 1
        }
    }
    return false
}
```

## 复杂度

| 指标 | 值 |
|------|-----|
| 时间 | O(log n) |
| 空间 | O(1) |
