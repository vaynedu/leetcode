# 栈与单调栈

## 识别信号

- 最近未匹配：括号、路径、表达式。
- 需要找左/右第一个更大或更小元素。
- 当前元素会让之前一些元素再也不可能成为答案。

## 核心不变量

- 普通栈：栈顶是最近还没被匹配或处理完的元素。
- 单调栈：栈内元素按值保持单调，存下标比存值更常用。
- 当新元素破坏单调性时，被弹出的元素此刻找到了自己的边界或答案。

## 决策流程

```text
1. 明确栈里存值还是下标。
2. 明确栈保持递增还是递减。
3. 遍历当前元素，while 当前元素破坏单调性则弹栈。
4. 弹栈时结算被弹元素答案。
5. 当前元素入栈，等待未来元素结算。
```

## 代表题

| # | 题目 | 类型 | 核心 |
|---|------|------|------|
| 20 | 有效括号 | 普通栈 | 最近左括号匹配 |
| 32 | 最长有效括号 | 栈下标 | 栈底保存合法段前一位 |
| 84 | 柱状图最大矩形 | 单调递增栈 | 弹栈时确定左右边界 |
| 42 | 接雨水 | 单调栈 | 弹出低洼柱计算水量 |

## 易错点

- 单调栈通常存下标，因为宽度和边界需要下标。
- 处理数组末尾时，常需要哨兵触发清算。
- 栈空时的宽度计算和 base index 最容易错。

## 为什么不会漏答案

元素留在栈中表示它右侧还没有出现能结算它的边界；一旦被弹出，当前元素就是第一个破坏单调性的右边界，左边界由新栈顶决定。
