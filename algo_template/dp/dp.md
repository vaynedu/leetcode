# 动态规划分类速查手册

## 识别信号

- 求最优值、计数、是否可达，且当前选择依赖之前结果。
- 暴力递归会重复计算相同子问题。
- 题目可以拆成前缀、网格坐标、容量、区间、集合状态或树形子问题。

## 核心不变量

- `dp` 的含义必须稳定：转移前后都表达同一件事。
- 每个状态只依赖已经算好的状态。
- 初始化覆盖最小子问题，遍历顺序保证依赖先于当前状态出现。
- 返回值必须和 `dp` 定义一致，不能临时猜。

## 决策流程

```text
1. 定义状态：dp[i] / dp[i][j] 到底表示什么。
2. 写选择：当前状态可以从哪些前置状态转移而来。
3. 写转移：max/min/sum/or。
4. 写初始化：空集、第一行、第一列、容量 0、长度 1。
5. 写遍历顺序：从小到大、从右到左、按区间长度、后序。
6. 确认答案位置：dp[n]、dp[n-1]、max(dp[i]) 或全局变量。
```

## 为什么不会漏答案

DP 枚举的是“状态空间”而不是原始路径。只要状态定义包含了影响未来决策的全部信息，转移枚举了所有可能选择，就不会漏掉最优解或计数。

## DP 解题checklist

```
1. 确定类型
   ├── 坐标型？→ 网格 DP
   ├── 序列型？→ dp[i] 结尾
   ├── 划分型？→ 分段计数
   ├── 背包？  → 选/不选
   ├── 区间型？→ 长度递增
   ├── 状态压缩？→ mask
   └── 树形？  → 后序

2. 定义 dp
   ├── dp[i] 或 dp[i][j] 的含义是什么？
   └── 结果在 dp[n-1] 还是 dp[...][...]？

3. 找状态转移
   ├── dp[i] = ?
   └── 找 dp[i] 与之前状态的关系

4. 初始化
   ├── dp[0] = ?
   └── 边界条件

5. 填表顺序
   └── for i from ? to ?
```

## 一、坐标型 DP

**特征**：网格/数组上移动，`dp[i][j]` 表示走到 (i,j) 的最优解

**核心公式**：
```
dp[i][j] = f(dp[i-1][j], dp[i][j-1])
```

**经典题**：
| 题号 | 标题 | 核心 |
|-----|------|------|
| 62 | 不同路径 | 计数 |
| 63 | 不同路径 II | 有障碍 |
| 64 | 最小路径和 | 带权重 |
| 174 | 地下城游戏 | 从后向前 |

---

## 二、序列型 DP

**特征**：`dp[i]` 表示以第 i 个元素结尾/截至 i 的最优解

**核心公式**：
```
dp[i] = max(dp[j] + ...) (j < i, 满足条件)
```

**经典题**：
| 题号 | 标题 | 核心 |
|-----|------|------|
| 300 | 最长上升子序列 | `nums[j] < nums[i]` |
| 139 | 单词划分 | 前缀匹配 |
| 91 | 解码方法 | `s[i-2:i]` 组合 |
| 198 | 打家劫舍 | `max(dp[i-1], dp[i-2]+nums[i])` |

---

## 三、划分型 DP

**特征**：将序列划分成若干段，取最值/计数

**核心公式**：
```
dp[i] = min(dp[j] + 1) (j < i, 满足条件)
```

**经典题**：
| 题号 | 标题 | 条件 |
|-----|------|------|
| 132 | 分割回文串 II | `isPalin[j+1][i]` |
| 410 | 分割数组的最大值 | `max(subSum) ≤ mid` |

---

## 四、背包 DP

**特征**：选或不选，容量限制，极值

| 类型 | 循环方向 | 核心公式 |
|-----|---------|---------|
| 01背包 | 从大到小 | `dp[j] = max(dp[j], dp[j-w] + v)` |
| 完全背包 | 从小到大 | `dp[j] = max(dp[j], dp[j-w] + v)` |

**经典题**：
| 题号 | 标题 | 类型 |
|-----|------|------|
| 416 | 分割等和子集 | 01 |
| 494 | 目标和 | 01计数 |
| 322 | 零钱兑换 | 完全 |
| 474 | 一和零 | 二维费用 |

---

## 五、区间型 DP

**特征**：处理区间 [i,j]，按长度递增递推

**核心公式**：
```
dp[i][j] = max(dp[i+1][j-1] + 2)  // 当 s[i]==s[j]
dp[i][j] = max(dp[i+1][j], dp[i][j-1])  // 当 s[i]!=s[j]
```

**经典题**：
| 题号 | 标题 | 核心 |
|-----|------|------|
| 516 | 最长回文子序列 | 两端相同+2 |
| 312 | 戳气球 | 区间划分+边界 |
| 877 | 石子游戏 | 区间博弈 |

---

## 六、状态压缩 DP

**特征**：用二进制 mask 表示集合状态

**核心公式**：
```
dp[mask][i] = min(dp[mask_without_i][j] + dist[j][i])
```

**适用场景**：TSP、集合覆盖、博弈

**经典题**：
| 题号 | 标题 | 核心 |
|-----|------|------|
| 464 | 我能赢吗 | 博弈+状压 |
| 698 | 划分为k个相等子集 | 集合划分 |

---

## 七、树形 DP

**特征**：在树上做 DP，用后序遍历

**核心公式**：
```
父节点 = f(左子树, 右子树)
```

**经典题**：
| 题号 | 标题 | 核心 |
|-----|------|------|
| 543 | 二叉树直径 | 左高+右高 |
| 124 | 二叉树最大路径和 | 后序+全局max |
| 337 | 打家劫舍 III | 选/不选子树 |

## 一图总结

```
┌─────────────────────────────────────────────────────────┐
│                      DP 类型速查                          │
├──────────┬──────────────┬──────────────────────────────┤
│ 坐标型   │ dp[i][j]     │ 网格移动，路径问题              │
│ 序列型   │ dp[i]        │ 以i结尾，最长上升子序列          │
│ 划分型   │ dp[i]        │ 分成若干段                      │
│ 背包     │ dp[j]        │ 选物品，容量限制                │
│ 区间型   │ dp[i][j]     │ 区间合并，先小后大              │
│ 状态压缩 │ dp[mask][i]  │ 集合表示，TSP/博弈             │
│ 树形     │ post-order   │ 后序遍历，父=合并子             │
└──────────┴──────────────┴──────────────────────────────┘
```
