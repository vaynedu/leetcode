二叉树的序列化与反序列化

将二叉树编码成字符串，并从字符串恢复二叉树

核心洞察

序列化：遍历二叉树 → 字符串。
反序列化：解析字符串 → 重构二叉树。
先序遍历（Preorder）最常用：根 → 左 → 右，唯一确定树结构。

序列化: "1,2,null,3,null,4,null,null,5"
反序列化: 读取字符串，重建同样的树结构

前序 Preorder

根 → 左 → 右
面试最常用

后序 Postorder

左 → 右 → 根
子节点信息在前

层序 Level Order

BFS 逐层
直观但需要队列

反序列化（Deserialize）

tokens: ["1","2","4","null","null","3","5","null","null","6","null","null"]
读取 1 → 创建根节点
读取 2 → 创建左子节点，递归处理 2 的子树
读取 4 → 左子，读取 null → 4 的左为空，读取 null → 4 的右为空
读取 3 → 回溯到根的右子节点，递归处理 3 的子树
逐个读取 token，前序顺序重建整棵树

Go 代码模板（前序遍历）

type TreeNode struct {
    Val   int
    Left  *TreeNode
    Right *TreeNode
}

// ---------- 序列化 ----------
func (s *Serializer) Serialize(root *TreeNode) string {
    var sb strings.Builder
    s.serializeHelper(root, &sb)
    return sb.String()
}

func (s *Serializer) serializeHelper(node *TreeNode, sb *strings.Builder) {
    if node == nil {
        sb.WriteString("null,")
        return
    }
    sb.WriteString(strconv.Itoa(node.Val))
    sb.WriteByte(',')
    s.serializeHelper(node.Left, sb)
    s.serializeHelper(node.Right, sb)
}

// ---------- 反序列化 ----------
func (s *Serializer) Deserialize(data string) *TreeNode {
    nodes := strings.Split(data, ",")
    idx := 0
    return s.deserializeHelper(nodes, &idx)
}

func (s *Serializer) deserializeHelper(nodes []string, idx *int) *TreeNode {
    if *idx >= len(nodes) {
        return nil
    }
    val := nodes[*idx]
    *idx++
    if val == "null" {
        return nil
    }
    intVal, _ := strconv.Atoi(val)
    node := &TreeNode{Val: intVal}
    node.Left = s.deserializeHelper(nodes, idx)
    node.Right = s.deserializeHelper(nodes, idx)
    return node
}

核心

前序遍历最常用

根、左、右的顺序，null 用作占位符

关键

索引指针传递

反序列化时用指针 idx 跟踪读取位置
