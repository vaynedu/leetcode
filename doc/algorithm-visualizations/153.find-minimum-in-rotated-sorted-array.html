寻找旋转排序数组中的最小值

找旋转点（最小值所在位置）

核心洞察

题目：在旋转排序数组中找到最小的元素。旋转排序数组原本是升序的，被旋转后分为两段升序。
关键：最小值一定在旋转点，即后半段升序的第一个元素。

旋转数组示例：

原数组: [1,2,3,4,5,6,7]

旋转 3: [4,5,6,7,1,2,3]

旋转点 = 4（最小值），前半段 [4,5,6,7] 比后半段 [1,2,3] 都大

nums[mid] > nums[R]？

↓

最小值在右半边
L = mid + 1

nums[mid] ≤ nums[R]？

↓

最小值在左半边（含mid）
R = mid

Go 代码模板

func findMin(nums []int) int {
    L, R := 0, len(nums)-1

    for L < R {  // 注意是 L < R，不是 L <= R
        mid := L + (R-L)/2

        if nums[mid] > nums[R] {
            // 最小值在右半边
            L = mid + 1
        } else {
            // nums[mid] <= nums[R]，最小值在左半边（含mid）
            R = mid
        }
    }
    return nums[R]  // L == R，最小值位置
}

为什么用 nums[R] 而不是 nums[L]？
因为原数组最后一个元素 nums[n-1] 是后半段升序的最大值，比较时更稳定。 用 nums[L] 也可以，但 nums[R] 更常用。
