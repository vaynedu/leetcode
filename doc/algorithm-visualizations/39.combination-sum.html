LeetCode 39 · Backtracking

组合总和

在有序数组中找和为目标值的组合，每个元素可重复使用

核心洞察

题目：给定无重复有序数组和目标值，找出所有和为目标值的组合。
关键： candidates 升序排列 + 每个元素可重复选 + 组合不重复。

与 #40 组合总和 II 的区别

#39: candidates 无重复，每个元素可用多次
#40: candidates 有重复，每个元素只用一次

去重方法

通过 index 参数控制起点，同一层不回头选之前的元素，自然去重

Go 代码模板

func combinationSum(candidates []int, target int) [][]int {
    sort.Ints(candidates)  // 排序，便于剪枝
    var result [][]int
    var path []int

    var backtrack func(start, target int)
    backtrack = func(start, target int) {
        if target == 0 {
            // 找到一组解
            ans := make([]int, len(path))
            copy(ans, path)
            result = append(result, ans)
            return
        }
        for i := start; i < len(candidates); i++ {
            if candidates[i] > target {
                // 剪枝：已排序，后面更大的数也不用试了
                break
            }
            path = append(path, candidates[i])
            backtrack(i, target-candidates[i])  // 关键：从 i 开始（可重复选自己）
            path = path[:len(path)-1]           // 回溯
        }
    }

    backtrack(0, target)
    return result
}

复杂度分析

时间复杂度

O(n^(k-1))

n=candidates长度，k=平均深度，最坏情况指数级

空间复杂度

O(k)

递归深度 = 解的长度

关键

index 参数

backtrack(i) 保证不回头选，自然去重

剪枝

排序 + 大于 target 跳过

排序后 candidates[i] > target 直接 break

与40区别

可重复 vs 不可重复

#39: backtrack(i)，#40: backtrack(i+1)
