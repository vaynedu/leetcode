LeetCode 17 · Backtracking

电话号码的字母组合

给定数字字符串，返回所有可能的字母组合

核心洞察

每个数字对应 3-4 个字母，求所有可能的组合，本质是多叉树的遍历问题。

2 abc
3 def
depth=0: 选择 "2" 的字母
a (digit[0]='2', letter='a')
b (digit[0]='2', letter='b')
c (digit[0]='2', letter='c')
depth=1: 选择 "3" 的字母
a → ad, ae, af
b → bd, be, bf
c → cd, ce, cf

手推全过程

digits = "23" → 结果: ["ad","ae","af","bd","be","bf","cd","ce","cf"]

初始 path = ""
""
Step 1 选择 '2' 的第 1 个字母 'a'
a
Step 2 选择 '3' 的字母: 'd' → 生成 "ad"
ad ae af
回溯 撤销 '3' 的选择，尝试下一个 'e'
ad ae af
回溯 撤销 '2' 的选择，尝试下一个 'b'
ad ae af bd
完成 共 9 种组合，全部生成
ad ae af bd be bf cd ce cf

Go 代码模板

var phone = map[byte]string{
    '2': "abc", '3': "def", '4': "ghi",
    '5': "jkl", '6': "mno", '7': "pqrs",
    '8': "tuv", '9': "wxyz",
}

func letterCombinations(digits string) []string {
    if len(digits) == 0 {
        return nil
    }
    var result []string
    var path []byte

    var backtrack func(idx int)
    backtrack = func(idx int) {
        if idx == len(digits) {
            result = append(result, string(path))
            return
        }
        letters := phone[digits[idx]]
        for i := 0; i < len(letters); i++ {
            path = append(path, letters[i])
            backtrack(idx + 1)
            path = path[:len(path)-1]  // 回溯：撤销选择
        }
    }

    backtrack(0)
    return result
}

复杂度分析

时间复杂度

O(4ⁿ × n)

n = digits 长度，每个数字最多 4 个字母

空间复杂度

O(n)

递归深度 = n（路径长度）

本质

多叉树遍历

每个数字的每个字母是一层分支

关键

回溯 = 选择 + 撤销

path.append() 后递归，然后 path.pop()

模板

index 控制深度

index == len(digits) 时收集结果
