Go 代码模板

方法一：拉平成一维数组

func searchMatrix(matrix [][]int, target int) bool {
    if len(matrix) == 0 || len(matrix[0]) == 0 {
        return false
    }
    m, n := len(matrix), len(matrix[0])
    L, R := 0, m*n-1

    for L <= R {
        mid := L + (R-L)/2
        row := mid / n
        col := mid % n
        if matrix[row][col] == target {
            return true
        } else if matrix[row][col] < target {
            L = mid + 1
        } else {
            R = mid - 1
        }
    }
    return false
}

方法二：从右上角搜索（更直观）

func searchMatrix2(matrix [][]int, target int) bool {
    if len(matrix) == 0 || len(matrix[0]) == 0 {
        return false
    }
    row := 0
    col := len(matrix[0]) - 1  // 右上角

    for row < len(matrix) && col >= 0 {
        if matrix[row][col] == target {
            return true
        } else if matrix[row][col] > target {
            col--  // 左移
        } else {
            row++  // 下移
        }
    }
    return false
}

两种方法对比

方法一：拉成一维

时间 O(log(m*n))
空间 O(1)
需要下标转换计算

方法二：右上角

时间 O(m+n)
空间 O(1)
更直观，易理解

面试建议：先说右上角方法（直观），被追问再说一维数组二分（更优雅）。
