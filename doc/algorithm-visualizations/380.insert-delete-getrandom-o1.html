LeetCode 380 · Design

O(1) 时间插入、删除和随机获取

实现一个支持 O(1) 平均时间的数据结构

核心洞察

题目：实现一个数据结构，支持：insert、delete、getRandom，且所有操作均为均摊 O(1)。
关键：HashMap + Array/List 的组合。

操作数据结构时间关键点
insertHashMap + ArrayO(1) map[val] = index, arr.append(val)
deleteHashMap + ArrayO(1) 与末尾交换，map 更新，arr.pop()
getRandomArrayO(1) rand.Intn(len(arr)) 随机索引

为什么 delete 是 O(1)？普通数组删除中间元素是 O(n)，但我们通过将被删除元素与末尾元素交换，然后 pop()，避免移动数组元素。

Go 代码模板

type RandomizedSet struct {
    // val → index 映射，O(1) 查找
    index map[int]int
    // 值列表，O(1) 随机访问
    vals  []int
}

func Constructor() RandomizedSet {
    return RandomizedSet{
        index: make(map[int]int),
        vals:  []int{},
    }
}

// 插入：O(1)
func (r *RandomizedSet) Insert(val int) bool {
    if _, exists := r.index[val]; exists {
        return false
    }
    r.index[val] = len(r.vals)
    r.vals = append(r.vals, val)
    return true
}

// 删除：O(1) — 关键是与末尾交换
func (r *RandomizedSet) Remove(val int) bool {
    idx, exists := r.index[val]
    if !exists {
        return false
    }
    // 将待删除元素与末尾交换
    last := len(r.vals) - 1
    lastVal := r.vals[last]
    r.vals[idx], r.vals[last] = r.vals[last], r.vals[idx]
    // 更新末尾元素的索引
    r.index[lastVal] = idx
    // 删除末尾
    r.vals = r.vals[:last]
    delete(r.index, val)
    return true
}

// 随机获取：O(1)
func (r *RandomizedSet) GetRandom() int {
    return r.vals[rand.Intn(len(r.vals))]
}

核心

HashMap + Array

Map 存索引，Array 存值，互补实现 O(1)

关键

末尾交换删除

避免数组中间删除 O(n)，改用交换+pop O(1)

要点

删除时更新索引

与末尾交换后，HashMap 中末尾元素索引要更新
