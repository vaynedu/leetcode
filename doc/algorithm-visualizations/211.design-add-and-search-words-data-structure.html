LeetCode 211 · Trie

添加与搜索单词

在 Trie 上支持通配符 '.' 搜索

核心洞察

题目：在 Trie 上支持 addWord(word) 和 search(word)，search 支持通配符 . 匹配任意单个字符。
关键：搜索时遇到 '.' 需要遍历所有子节点（而不是只走一个），这使得最坏时间复杂度为 O(m × 26)。

普通字符

精确匹配，只走对应子节点

通配符 '.'

匹配任意字符，遍历所有子节点

手推全过程

添加: ["bad", "dad", "mad"]，搜索: "pad" 和 "b.d"

搜索 "pad"
p
a
d
从 root 开始，走 'p' → 节点不存在
返回 false（'pad' 不在字典中）
搜索 "b.d"
b
.
d
root → 'b' → 存在
遇到 '.'，需要尝试 'b' 的所有子节点
'b' 有子节点 'a'，'a' 有子节点 'd'（bad）
"b.d" 匹配 "bad" → 返回 true

Go 代码模板

type WordDictionary struct {
    root *TrieNode
}

func Constructor() WordDictionary {
    return WordDictionary{root: NewTrieNode()}
}

func (w *WordDictionary) AddWord(word string) {
    node := w.root
    for _, ch := range word {
        if _, ok := node.children[ch]; !ok {
            node.children[ch] = NewTrieNode()
        }
        node = node.children[ch]
    }
    node.isEnd = true
}

func (w *WordDictionary) Search(word string) bool {
    return w.dfs(word, 0, w.root)
}

func (w *WordDictionary) dfs(word string, index int, node *TrieNode) bool {
    if index == len(word) {
        return node.isEnd
    }
    ch := rune(word[index])

    if ch == '.' {
        // 遍历所有子节点
        for _, child := range node.children {
            if w.dfs(word, index+1, child) {
                return true
            }
        }
        return false
    }

    // 精确匹配
    if child, ok := node.children[ch]; ok {
        return w.dfs(word, index+1, child)
    }
    return false
}

复杂度分析

时间复杂度

O(m × 26^k)

m=词长，k='.'个数，最坏指数级

空间复杂度
