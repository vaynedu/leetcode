LEETCODE 34 · FIND FIRST AND LAST POSITION

二分查找边界

左边界找"第一个 ≥ target" · 右边界找"第一个 > target"

💡 核心思路
找左边界：right=mid；找右边界：left=mid+1，最后用左右指针收尾
口诀：左边界 right=mid，右边界 left=mid+1

① 两个边界是什么

在排序数组中，target 可能出现多次。左边界是第一个 ≥ target 的位置，右边界是最后一个 ≤ target 的位置。

答案
target 出现两次
边界目标收缩条件区间
左边界 first ≥ target nums[mid] < target → left=mid+1
else → right=mid		 [left, right)
右边界 first > target nums[mid] ≤ target → left=mid+1
else → right=mid		 [left, right)
唯一区别：等号往哪边走。
左边界：nums[mid] < target 才 left+1（= 时往左缩）
右边界：nums[mid] ≤ target 才 left+1（= 时往右扩）

④ 两个边界一起看

左边界和右边界的区别只在一个符号。左边界的 "=" 往左走，右边界的 "=" 往右走。

● 左边界 lowerBound(8)
条件：nums[mid] < target
7 < 8 → left = mid+1 → 往右缩
8 < 8 → 否 → right = mid → 往左缩
退出 left=3
leftBound = 3
● 右边界 lowerBound(9) − 1
条件：nums[mid] ≤ target
7 ≤ 8 → left = mid+1 → 往右扩
10 ≤ 8 → 否 → right = mid → 往左缩
8 ≤ 8 → left = mid+1 → 往右扩
退出 left=5 → −1
rightBound = 4
最终结果
[3, 4]
nums[3]=8, nums[4]=8

左边界

first ≥ target

nums[mid] < target → left=mid+1
else → right=mid

右边界

first > target

nums[mid] ≤ target → left=mid+1
else → right=mid
