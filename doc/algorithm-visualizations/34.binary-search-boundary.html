LeetCode 34 · Binary Search Boundary

查找第一个和最后一个位置

把“找 target”转成两次 lowerBound：第一个不小于 target，和第一个大于 target。

核心洞察

题目：排序数组中找 target 的起始和结束位置。
关键：不要找到任意 target 就停；先找左边界 lowerBound(target)，再找右边界 lowerBound(target+1)-1。

观察

边界比命中更重要

普通二分只能找到某个 target，不能保证是第一个或最后一个。

不变量

答案始终在 [left,right)

左闭右开区间里，right 不参与搜索，收缩时可以写 right=mid。

复杂度

O(log n) 时间 O(1) 空间

两次二分，每次都把搜索区间缩小一半。

手推全过程

样例：nums=[5,7,7,8,8,10]，target=8

Left 1 lowerBound(8), [0,6), mid=3
nums[3]=8 >= 8 mid 可能就是第一个 8 right = mid = 3
找左边界时，等于 target 也要往左缩，不能越过第一个 target。
Left 2 [0,3), mid=1
nums[1]=7 < 8 mid 和左边都不可能是答案 left = mid + 1 = 2
继续收缩后最终得到 lowerBound(8)=3。
Right lowerBound(9)
找第一个 >= 9 的位置 lowerBound(9)=5 rightBound = 5 - 1 = 4
右边界不是单独写一套复杂逻辑，而是“第一个大于 target 的位置减一”。

Go 代码

复用一个 lowerBound，避免左右边界写两套容易出错的二分。

func SearchRange(nums []int, target int) []int {
    if len(nums) == 0 {
        return []int{-1, -1}
    }

    left := lowerBound(nums, target)
    if left == len(nums) || nums[left] != target {
        return []int{-1, -1}
    }

    right := lowerBound(nums, target+1) - 1
    return []int{left, right}
}

func lowerBound(nums []int, target int) int {
    left, right := 0, len(nums) // [left, right)
    for left < right {
        mid := left + (right-left)/2
        if nums[mid] < target {
            left = mid + 1
        } else {
            right = mid // mid 可能就是第一个 >= target
        }
    }
    return left
}

易错点

边界二分最容易错在“找到就返回”和“左闭右开里丢掉 mid”。

错误：命中 target 就返回

if nums[mid] == target {
    return mid
}

这只能找到某个 8，不能保证同时得到最左和最右边界。

正确：继续压缩到边界

if nums[mid] < target {
    left = mid + 1
} else {
    right = mid
}

即使命中 target，mid 也可能不是第一个 target，要保留它继续往左找。

错误：左闭右开写 right=mid-1

if nums[mid] >= target {
    right = mid - 1
}

在 [left,right) 模板里，mid 可能就是答案，减一会把答案丢掉。

正确：保留 mid

if nums[mid] >= target {
    right = mid
}

right 是开区间边界，收缩到 mid 才能保留候选答案。

核心算法

两次 lowerBound

左边界是 lowerBound(target)，右边界是 lowerBound(target+1)-1。

正确性直觉

命中也不停止

边界搜索要找第一个满足条件的位置，命中 target 时仍要收缩。

易错关键点

[left,right) 保留 mid

nums[mid] >= target 时写 right=mid，不能写 mid-1。
