Go 代码模板

func permute(nums []int) [][]int {
    var result [][]int
    used := make([]bool, len(nums))
    var path []int

    var backtrack func()
    backtrack = func() {
        if len(path) == len(nums) {
            ans := make([]int, len(path))
            copy(ans, path)
            result = append(result, ans)
            return
        }
        for i := 0; i < len(nums); i++ {
            if used[i] {
                continue  // 已使用的跳过
            }
            used[i] = true
            path = append(path, nums[i])
            backtrack()
            path = path[:len(path)-1]
            used[i] = false  // 回溯
        }
    }

    backtrack()
    return result
}

复杂度分析

时间复杂度

O(n × n!)

共 n! 个排列，复制路径 O(n)

空间复杂度

O(n)

used 数组 + 递归深度

本质

枚举每个位置

每个位置从所有未使用的数中选一个

关键

used 数组

标记已使用的元素，避免重复选

模板

index 参数无关

全排列不看 index，只看 used 状态
