设计井字棋

实现一个支持 move、check 操作的井字棋裁判

核心洞察

题目：实现一个 3x3 井字棋游戏，支持 move(row, col, player) 和 check() 判断胜者。
关键：用数组替代 3x3 矩阵，每次 move 更新行/列/对角线计数，check 时 O(1) 判断。

普通做法

每次 move 遍历所有行/列/对角线
O(9) = O(1) 常数，但不够优雅

优化做法（推荐）

row[3] + col[3] + diag[2]
每次 move 只更新对应行/列/对角
check 时 O(1)

核心数据结构

row[3]

每行累计差值
Player1 下棋 +1，Player2 下棋 -1
某行 |sum| == 3 → 该玩家赢

col[3]

每列累计差值
同理，某列 |sum| == 3 → 赢

diag[2]

主对角线 + 副对角线
diag[0]: 主对角线，diag[1]: 副对角线
|sum| == 3 → 赢

手推全过程

Player1 (X) 先手，对局：
P1:(0,0), P2:(1,0), P1:(0,1), P2:(1,1), P1:(0,2) → P1 赢（行 0）

Go 代码模板

type TicTacToe struct {
    n    int
    // 行累计差值：Player1下棋+1，Player2下棋-1
    rows []int
    // 列累计差值
    cols []int
    // 对角线累计差值: [0]主对角线 [1]副对角线
    diags []int
}

func Constructor(n int) TicTacToe {
    return TicTacToe{
        n:     n,
        rows:  make([]int, n),
        cols:  make([]int, n),
        diags: make([]int, 2),
    }
}

// move: O(1)
func (t *TicTacToe) Move(row, col, player int) int {
    delta := 1
    if player == 2 {
        delta = -1
    }
    t.rows[row] += delta
    t.cols[col] += delta
    if row == col {
        t.diags[0] += delta
    }
    if row+col == t.n-1 { // 副对角线: row + col == n-1
        t.diags[1] += delta
    }
    // 检查是否获胜
    if abs(t.rows[row]) == t.n || abs(t.cols[col]) == t.n ||
        abs(t.diags[0]) == t.n || abs(t.diags[1]) == t.n {
        return player
    }
    return 0 // 无胜者
}

func abs(x int) int {
    if x < 0 {
        return -x
    }
    return x
}

核心

差值计数法

Player1 +1，Player2 -1，|sum| == n 则获胜

时间

move O(1)

只更新行、列、对角线，无须遍历整盘棋

扩展

n x n 通用

构造函数传入 n，可推广到任意大小棋盘
