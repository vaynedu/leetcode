顶部迭代器

设计一个能同时迭代两个有序链表的迭代器

核心洞察

题目：给定两个有序链表，实现 PeekingIterator，除了常规的 next/hasNext，还要支持 peek() 查看下一个元素而不移动指针。
关键：缓存下一个元素，用 peek 直接返回，next 时补充新缓存。

iterator = PeekingIterator([1, 2, 3])
iterator.next() → 1（常规）
iterator.peek() → 2（不移动！）
iterator.next() → 2
iterator.next() → 3

Go 代码模板

type PeekingIterator struct {
    iter  *list.Iterator
    // 缓存下一个元素
    peekCache int
    // 缓存是否有效
    hasPeek   bool
}

func Constructor(iter *list.Iterator) *PeekingIterator {
    pi := &PeekingIterator{
        iter:    iter,
        hasPeek: iter.Next(),
    }
    if pi.hasPeek {
        pi.peekCache = iter.Value.(int)
    }
    return pi
}

func (it *PeekingIterator) Peek() int {
    if !it.hasPeek {
        panic("no more elements")
    }
    return it.peekCache
}

func (it *PeekingIterator) Next() int {
    if !it.hasPeek {
        panic("no more elements")
    }
    val := it.peekCache
    it.hasPeek = it.iter.Next()
    if it.hasPeek {
        it.peekCache = it.iter.Value.(int)
    }
    return val
}

func (it *PeekingIterator) HasNext() bool {
    return it.hasPeek
}

核心

peek 缓存

额外维护一个 peekCache 变量，peek 直接返回

关键

next 补充缓存

每次 next 返回缓存后，立即从底层 iterator 补充新缓存
