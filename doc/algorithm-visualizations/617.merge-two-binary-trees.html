LeetCode 617 · Tree DFS + Merge

合并二叉树

同步递归两棵树：双非空节点值相加，单边为空时直接保留另一棵子树。

核心洞察

题目：把两棵二叉树按相同位置合并。
关键：不是把 nil 当作值 0 创建新节点，而是某一边为空时直接返回另一边整棵子树。

观察

同位置合并

两个节点都存在时，合并值为 root1.Val + root2.Val。

选择

单空直接返回非空

如果一个位置只有一棵树有节点，结果应该保留这整段子树。

复杂度

O(k) 时间 O(h) 空间

k 为递归访问到的重叠区域节点数；当前实现原地修改 root1。

手推全过程

样例：root1=[1,3,2,5]，root2=[2,1,3,null,4,null,7]

Step 1 root: 1 + 2 = 3
root1=1 root2=2 merged=3
根节点两边都存在，直接把值相加，写回 root1。
结果根节点为 3。
Step 2 left: 3 + 1 = 4
root1.Left=3 root2.Left=1 merged=4
左子树根也双非空，继续值相加。
左子树根变为 4。
Step 3 single side: 5 + nil, nil + 4
5 + nil → keep 5 nil + 4 → keep 4
左子树继续向下时，有些位置只有一棵树有节点。
直接返回非空节点，不能丢掉这段子树。
Step 4 right: 2 + 3 = 5, nil + 7 = 7
2 + 3 → 5 nil + 7 → keep 7
右子树根相加；右右节点只存在于 root2。
最终结果层序为 [3,4,5,5,4,null,7]。

Go 代码

原地合并版本：返回值要重新接回左右指针。

func MergeTrees(root1, root2 *TreeNode) *TreeNode {
    if root1 == nil {
        return root2
    }
    if root2 == nil {
        return root1
    }

    root1.Val += root2.Val
    root1.Left = MergeTrees(root1.Left, root2.Left)
    root1.Right = MergeTrees(root1.Right, root2.Right)
    return root1
}

易错点

单空位置不能返回 nil；这会丢掉另一棵树本来存在的节点。

错误：单空返回 nil

if root1 == nil || root2 == nil {
    return nil
}

输入 root1=[1], root2=nil 时，正确结果是 [1]，不是空树。

正确：返回非空子树

if root1 == nil {
    return root2
}
if root2 == nil {
    return root1
}

单边存在时，另一棵树没有对应节点，结果应完整保留非空子树。

核心算法

同步 DFS

同位置递归合并两棵树，双非空相加，单非空直接返回。

正确性直觉

每个位置独立合并

当前节点的合并结果只依赖当前位置两棵子树根和左右子问题。

易错关键点

保留非空子树

单空位置直接返回另一边，避免丢掉只存在于一棵树的节点。
