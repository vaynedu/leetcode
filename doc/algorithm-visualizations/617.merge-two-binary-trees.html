合并二叉树

相同位置节点值相加，返回合并后的树

💡 核心思路
同步前序遍历：两树节点相加，nil 当 0，缺一树直接复用
口诀：相加，nil 当 0

三棵树对比

Tree 2

Merged

1+2=3, 3+1=4, 2+3=5

原地合并代码

func MergeTrees(root1, root2 *TreeNode) *TreeNode { // 其中一个为空，直接返回另一个 if root1 == nil { return root2 } if root2 == nil { return root1 } // 都有值：相加 root1.Val += root2.Val // 递归合并左右子树 root1.Left = MergeTrees(root1.Left, root2.Left) root1.Right = MergeTrees(root1.Right, root2.Right) return root1 }

合并顺序

先处理根，再递归处理左子树和右子树。返回值直接接在当前节点的 Left/Right 上。

root1==nil → return root2 → root2==nil → return root1 → Val相加
