LeetCode 179 · Sort

最大数

给定非负整数数组，返回能拼接出的最大数字

核心洞察

题目：给定数组 [10, 2]，拼接最大数为 "210"（不是 "102"）。
关键：自定义排序比较：a+b 和 b+a 哪个更大。

"3" vs "30" "330" vs "303" "330" ✓
"9" vs "98" "998" vs "989" "998" ✓
"121" vs "12" "12112" vs "12121" "12121" ✓

排序规则：如果 a+b > b+a，则 a 排在前面
例如：a="3", b="30" → "330" > "303" → 3 在前
核心：把数字转字符串，用字符串比较替代数字比较

手推全过程

输入: [10, 2, 9, 30, 3, 5]

Step 1 两两比较，排序
"30" vs "3": "303" < "330" → "3" 在前
"30" vs "9": "309" < "930" → "9" 在前
"10" vs "2": "102" < "210" → "2" 在前
"10" vs "30": "1030" < "3010" → "30" 在前
"9" vs "5": "95" > "59" → "9" 在前
Step 2 排序结果: ["9","3","5","30","2","10"]
拼接: "9"+"3"+"5"+"30"+"2"+"10" = "95330210"

Go 代码模板

import (
    "sort"
    "strconv"
    "strings"
)

func largestNumber(nums []int) string {
    // 将所有数字转字符串
    strs := make([]string, len(nums))
    for i, n := range nums {
        strs[i] = strconv.Itoa(n)
    }

    // 自定义排序：a+b > b+a 则 a 在前
    sort.Slice(strs, func(i, j int) bool {
        return strs[i]+strs[j] > strs[j]+strs[i]
    })

    // 拼接
    var sb strings.Builder
    for _, s := range strs {
        sb.WriteString(s)
    }

    result := sb.String()
    // 排除全0情况
    if result[0] == '0' {
        return "0"
    }
    return result
}

特殊情况：全 0

如果输入是 [0, 0]，拼接结果是 "00"，需要返回 "0"。

排序后: ["0","0"]，拼接: "00"
检查 result[0] == '0' → 返回 "0"

核心

字符串拼接排序

a+b > b+a → a 排在前面，找最大数字

陷阱

全 0 返回 "0"

"00" 需特殊处理，避免前导零

扩展

最小数

改为 a+b < b+a 即可得到最小拼接数
