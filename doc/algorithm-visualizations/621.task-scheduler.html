621. 任务调度器

贪心：先排最多数量的任务，形成 idle 间隔填入其他任务

阶段: -
执行任务: -
时间片: -
冷却计时: -
点击「下一步」开始
总时间: - 空闲数: - 
function leastInterval(tasks, n) {
  const cnt = {};
  tasks.forEach(t => cnt[t] = (cnt[t]||0)+1);
  const max = Math.max(...Object.values(cnt));
  const maxCnt = Object.values(cnt).filter(v=>v===max).length;
  return Math.max(tasks.length, (max-1)*(n+1)+maxCnt);
}
