二叉树的所有路径

Root-to-leaf paths, "1→2→5" format

💡 核心思路
前序遍历，路径字符串拼接，到叶子节点收集
口诀：前序收集，字符串累积

问题定义

返回所有从根到叶子节点的路径，格式为 1->2->5。

递归过程

用 path slice 记录当前路径，叶子节点时拼接字符串收集。

根1path=[1]，不是叶子，递归 Left(2) 和 Right(3)
节点2path=[1,2]，不是叶子，递归 Left(5) 和 Right(6)
节点5path=[1,2,5]，是叶子！收集 1->2->5
节点6path=[1,2,6]，是叶子！收集 1->2->6
节点3path=[1,3]，是叶子！收集 1->3
→ 1->2->5
→ 1->2->6
→ 1->3

模板口诀

path 传递

切片值拷贝，自动回溯，无需手动删除

叶子收集

左右子树都为nil时，拼接字符串存入结果

递归框架

加入当前→递归左右→（叶子时收集）
