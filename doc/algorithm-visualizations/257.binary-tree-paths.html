LeetCode 257 · Tree DFS + Path

二叉树的所有路径

DFS 维护根到当前节点的路径，到叶子时拼成字符串并收集。

核心洞察

题目：返回所有从根到叶子的路径，格式为 "1->2->5"。
关键：只有叶子节点才形成完整路径；path 是递归状态，不是最终答案。

观察

根到叶才收集

中间节点只是路径前缀，必须继续向下直到左右孩子都为空。

选择

path 随 DFS 传递

每层 append 当前值，递归左右子树；当前实现用切片值传递，天然回到上一层路径。

复杂度

O(n*h) 时间 O(h) 空间

访问每个节点；叶子处拼接字符串需要当前路径长度，最长为树高 h。

手推全过程

样例：root=[1,2,3,null,5]

Step 1 path=[1,2] node=2 not leaf
1→ 2
节点 2 不是叶子，因为它还有右孩子 5。
不能收集 "1->2"，必须继续走到 5。
Step 2 path=[1,2,5] node=5 leaf
1→ 2→ 5
节点 5 左右孩子都为空，是叶子节点。
收集 "1->2->5"。
Step 3 path=[1,3] node=3 leaf
1→ 3
回溯到根节点后递归右子树 3，3 是叶子。
最终答案：["1->2->5","1->3"]。

Go 代码

DFS 维护 path，叶子处拼接路径字符串。

import "strconv"

func binaryTreePaths(root *TreeNode) []string {
    ans := []string{}
    path := []int{}

    var dfs func(*TreeNode)
    dfs = func(node *TreeNode) {
        if node == nil {
            return
        }
        path = append(path, node.Val)

        if node.Left == nil && node.Right == nil {
            ans = append(ans, formatPath(path))
        } else {
            dfs(node.Left)
            dfs(node.Right)
        }

        path = path[:len(path)-1] // 显式回溯
    }

    dfs(root)
    return ans
}

func formatPath(path []int) string {
    s := strconv.Itoa(path[0])
    for i := 1; i < len(path); i++ {
        s += "->" + strconv.Itoa(path[i])
    }
    return s
}

易错点

最常见错误是提前收集半截路径，或用只支持个位数的字符转换。

错误：非叶子也收集

path = append(path, node.Val)
ans = append(ans, formatPath(path))

样例中会错误收集 "1" 和 "1->2"，但它们都不是根到叶路径。

正确：只在叶子收集

if node.Left == nil && node.Right == nil {
    ans = append(ans, formatPath(path))
}

叶子节点才代表完整路径；多位数或负数应使用 strconv.Itoa。

核心算法

DFS + path

每层把当前节点加入 path，递归左右子树，回溯时撤销当前节点。

正确性直觉

每个叶子收集一次

DFS 会覆盖所有根到叶路径，且只在叶子节点把当前 path 转成字符串。

易错关键点

不要收集前缀

非叶子路径只是前缀；转换数字时要支持多位数和负数。
