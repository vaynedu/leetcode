LeetCode 353 · Design

设计贪吃蛇游戏

实现一个支持 move(direction) 的贪吃蛇游戏

核心洞察

题目：设计一个 m×n 棋盘上的贪吃蛇游戏，支持 move(direction)，返回蛇是否活着（没撞墙/没撞自己）。
关键：用 deque（双端队列）表示蛇身体，头部入队，尾部出队。

蛇身 deque

队列头部=蛇头，尾部=蛇尾
移动：头入，尾出（不吃到食物时）

食物位置

Set 或 HashMap 存食物坐标
吃到食物时从 Set 移除，新食物随机生成

碰撞检测

新头部坐标撞到墙→死
新头部坐标在 deque 中→撞自己死

手推全过程

棋盘 3×3，初始蛇在 (1,1)，食物在 (2,2)，执行 move "U", "R", "D"

初始状态

S
F

蛇: [(1,1)] · 食物: {(2,2)}

Op 1 move("U") → true（活着）
新头部 = (0, 1)，不在 deque，不撞墙
(0,1) ≠ (2,2)，不吃食物
蛇尾 (1,1) 弹出，蛇头 (0,1) 入队 → 蛇: [(0,1)]
Op 2 move("R") → true（活着）
新头部 = (0, 2)
(0,2) ≠ (2,2)，不吃食物
蛇尾 (0,1) 弹出，蛇头 (0,2) 入队 → 蛇: [(0,2)]
Op 3 move("D") → true（吃到食物！）
新头部 = (1, 2)
(1,2) == 食物 (2,2)? 否...等等，重新看：食物在 (2,2)，新头在 (1,2)
继续：实际食物是 (2,2)，新头是 (1,2)... 让我重新说明：
如果食物在 (1,2)：(1,2) == (1,2) ✓ 吃到！
吃到食物：蛇尾不出队，食物从 Set 移除，随机生成新食物

Go 代码模板

type SnakeGame struct {
    width, height int
    // 蛇身体，头部在队列前端
    snake [][]int
    // 食物位置集合
    food map[int]bool
    // 食物列表（按顺序）
    foods [][]int
    foodIdx int
}

func Constructor(width, height int, food [][]int) SnakeGame {
    return SnakeGame{
        width:  width,
        height: height,
        snake:  [][]int{{0, 0}}, // 初始蛇在 (0,0)
        food:   make(map[int]bool),
        foods:  food,
        foodIdx: 0,
    }
}

func (s *SnakeGame) Move(direction string) bool {
    head := s.snake[0]
    newHead := []int{head[0], head[1]}
    switch direction {
    case "U": newHead[0]--
    case "D": newHead[0]++
    case "L": newHead[1]--
    case "R": newHead[1]++
    }

    // 撞墙
    if newHead[0] < 0 || newHead[0] >= s.height ||
        newHead[1] < 0 || newHead[1] >= s.width {
        return false
    }

    // 撞自己（排除尾巴，因为尾巴会移动）
    for i := 0; i < len(s.snake)-1; i++ {
        if s.snake[i][0] == newHead[0] && s.snake[i][1] == newHead[1] {
            return false
        }
    }

    // 吃到食物
    ate := s.foodIdx < len(s.foods) &&
        newHead[0] == s.foods[s.foodIdx][0] &&
        newHead[1] == s.foods[s.foodIdx][1]

    if ate {
        s.foodIdx++
    } else {
        // 没吃到：尾巴移动
        s.snake = s.snake[1:]
    }

    // 新头部入队
    s.snake = append([][]int{newHead}, s.snake...)
    return true
}

核心

deque 存蛇身

头部入队，尾部出队（吃到食物时不出队）

关键

撞自己判断时机

判断撞自己要排除尾巴，因为尾巴会移动出队

变种

变长/无限棋盘

无限模式用 Set 代替固定二维数组
