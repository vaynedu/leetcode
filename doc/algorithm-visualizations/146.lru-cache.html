操作复杂度

操作 HashMap 链表
Get O(1) 查找 O(1) 移到头
Put O(1) 插入/删除 O(1) 头部插入+尾部删除

为什么双向链表？

删除一个节点需要知道它的前驱。
单向链表找前驱要 O(n)，双向链表直接 node.prev.next = node.next O(1)。
