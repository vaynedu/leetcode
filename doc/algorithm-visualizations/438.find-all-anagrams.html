固定窗口滑动，字母计数数组对比

核心洞察

题目：给定字符串 s 和 p，找出 s 中所有 p 的异位词起始下标。
异位词 = 字母相同但排列不同的词，如 "abc" 和 "bca"。
关键：固定窗口大小，滑动时用计数数组判断两窗口是否字母组成相同。

怎么判断两个字符串字母组成相同？
用一个 26 长度数组记录窗口内每个字母的出现次数，和模式串 p 的计数对比。
如果 diff == 0（所有字母计数差为 0），则当前位置是异位词。

Go 代码模板

固定窗口大小 len(p)，用 diff=0 判断是否匹配

func findAnagrams(s, p string) []int {
    need := [26]int{}
    for _, c := range p {
        need[c-'a']++
    }

    res := []int{}
    diff := 0  // 还有几个字母未匹配

    for i := range s {
        // 入窗口
        idx := s[i] - 'a'
        if need[idx] > 0 {
            diff++ // need 中有的字母才计入 diff
        }
        need[idx]-- // 计数相减

        // 出窗口
        if i >= len(p) {
            outIdx := s[i-len(p)] - 'a'
            if need[outIdx] >= 0 {
                diff-- // 移出的字母原本是匹配的
            }
            need[outIdx]++
        }

        if diff == 0 {
            res = append(res, i-len(p)+1)
        }
    }
    return res
}

diff 表示"窗口内字母"比"模式串字母"多出现了几次（need>0的部分）
