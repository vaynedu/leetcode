路径总和 II

收集所有满足条件的根到叶子路径

问题对比

112：判断是否存在 → 返回 bool
113：收集所有路径 → 返回 [][]int

收集所有路径

func PathSumII(root *TreeNode, target int) [][]int { var result [][]int var path []int helper(root, target, path, &result) return result } func helper(node *TreeNode, target int, path []int, result *[][]int) { if node == nil { return } path = append(path, node.Val) // 加入当前节点 if node.Left == nil && node.Right == nil { if node.Val == target { // 叶子且值匹配 tmp := make([]int, len(path)) copy(tmp, path) // ⚠ 必须 copy! *result = append(*result, tmp) } return } helper(node.Left, target - node.Val, path, result) helper(node.Right, target - node.Val, path, result) }

Trace：target=22

5→4target 22-5=17，递归左子树4
4→11target 17-4=13，递归左子树11
11→2 (叶!)target 13-11=2，2是叶子且值=2 → 匹配！收集 [5,4,11,2]
回溯到4继续递归右子树13，13不是叶子，target 17-4=13
13→nil13是叶子但值≠13，无结果

与 112 的关键区别

返回值

112 返回 bool（找到即停），113 返回 [][]int（收集所有）

路径保存

叶子匹配时必须 copy(path)，否则后续回溯会修改

递归结构

左右子树都要递归，不能短路返回
