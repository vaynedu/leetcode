LeetCode 113 · Tree DFS + Backtracking

路径总和 II

DFS 枚举所有根到叶路径，叶子处判断剩余目标，命中时复制当前 path。

核心洞察

题目：返回所有从根到叶子且路径和等于 targetSum 的路径。
关键：路径必须到叶子才完整；收集答案时必须复制 path，不能保存可变切片本身。

观察

根到叶才算一条路径

中途节点即使当前和等于 target，也不能提前收集，必须继续走到叶子。

选择

递减 target 比累计 sum 更直接

进入子节点时传 target-node.Val，到叶子只需判断 node.Val == target。

复杂度

O(n*h) 时间 O(h) 空间

每个节点访问一次；每次收集路径要复制当前 path，最长长度为树高 h。

手推全过程

样例：root=[5,4,8,11,null,13,4,7,2,null,null,5,1]，targetSum=22

Step 1 path=[5,4,11,7] target=2 leaf=7
5→ 4→ 11→ 7
走到叶子 7 时，前面已经扣掉 5、4、11，所以剩余 target 是 2。
7 != 2，路径 [5,4,11,7] 不收集。
Step 2 path=[5,4,11,2] target=2 leaf=2
5→ 4→ 11→ 2
走到叶子 2 时，剩余 target 正好是 2。
收集 [5,4,11,2]，注意要 copy(path)。
Step 3 path=[5,8,13] target=9 leaf=13
5→ 8→ 13
右子树左叶子 13：剩余 target 为 9。
13 != 9，不收集。
Step 4 path=[5,8,4,5] target=5 leaf=5
5→ 8→ 4→ 5
路径 [5,8,4,5] 的和也是 22。
最终答案：[[5,4,11,2],[5,8,4,5]]。

Go 代码

递归时 path 作为当前路径前缀；叶子命中时复制后加入结果。

func PathSumII(root *TreeNode, targetSum int) [][]int {
    var result [][]int
    var path []int
    pathSumRec(root, targetSum, path, &result)
    return result
}

func pathSumRec(node *TreeNode, target int, path []int, result *[][]int) {
    if node == nil {
        return
    }

    path = append(path, node.Val)

    if node.Left == nil && node.Right == nil {
        if node.Val == target {
            tmp := make([]int, len(path))
            copy(tmp, path) // 必须复制当前路径
            *result = append(*result, tmp)
        }
        return
    }

    pathSumRec(node.Left, target-node.Val, path, result)
    pathSumRec(node.Right, target-node.Val, path, result)
}

易错点

这题最容易把“路径定义”和“可变 path”搞错。

错误：非叶子节点就收集

path = append(path, node.Val)
if node.Val == target {
    result = append(result, path)
    return
}

输入 root=[1,2], target=1 时，根节点 1 不是叶子，不能收集 [1]。

正确：只在叶子判断并复制

if node.Left == nil && node.Right == nil {
    if node.Val == target {
        tmp := make([]int, len(path))
        copy(tmp, path)
        result = append(result, tmp)
    }
}

叶子才代表一条完整根到叶路径；复制可以避免后续递归污染已保存答案。

核心算法

DFS + path

递归维护根到当前节点路径，同时把 target 逐层扣减。

正确性直觉

每个叶子检查一次

所有根到叶路径都会被 DFS 覆盖，且只在叶子处判断完整路径。

易错关键点

收集时 copy

path 是递归状态，命中答案时必须复制，不能保存后续会变化的底层数组。
