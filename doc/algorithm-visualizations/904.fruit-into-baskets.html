Go 代码模板

用 map 记录每种水果的数量，>2 种时收缩左指针

func totalFruit(trees []int) int {
    count := make(map[int]int)
    L := 0
    maxLen := 0

    for R := 0; R < len(trees); R++ {
        count[trees[R]]++  // 入窗口

        // 超过两种，收缩左指针
        for len(count) > 2 {
            count[trees[L]]--
            if count[trees[L]] == 0 {
                delete(count, trees[L])
            }
            L++
        }

        maxLen = max(maxLen, R-L+1)
    }
    return maxLen
}

变形与关键

本题本质是"最多 K 种"的变长窗口特例（K=2）

本题（K=2）

最多两种水果
用 map 记录每种数量
超过 2 种时收缩 L

扩展（K=3）

最多三种水果
同样逻辑
只需改 len(count) > 3

内层 while 收缩的正确性：
每次只移出 L 一个元素，直到 count 里恰好 ≤2 种。收缩过程中 count 会依次删掉第一种直到只剩两种。

类型

变长窗口

R 扩展，超过 2 种时内层 while 收缩 L

数据结构

map 计数

key=水果类型，value=出现次数，len(map) 即种类数

核心

len(count) ≤ 2

窗口内种类数始终 ≤2，每次更新 maxLen
