搜索插入位置

在有序数组中找 target，找不到则返回插入位置

核心洞察

题目：给定有序数组 nums 和目标值 target，如果存在则返回下标，否则返回插入位置。
关键：标准二分查找，找不到时返回 L（恰好是插入位置）。

为什么 L 就是插入位置？
二分结束时 L==R，落在 target 应该所在的"槽位"。如果 target 存在，命中时会返回 mid； 如果不存在，循环结束时 L 停在第一个 > target 的位置（或者数组末尾）。

Go 代码模板

标准二分，注意循环条件和返回值

func searchInsert(nums []int, target int) int {
    L, R := 0, len(nums)-1

    for L <= R {
        mid := L + (R-L)/2
        if nums[mid] == target {
            return mid
        } else if nums[mid] < target {
            L = mid + 1
        } else {
            R = mid - 1
        }
    }
    // 找不到时，L 就是插入位置
    return L
}

易错点

❌ 循环条件 L < R

如果用 L < R，可能漏掉 target 恰好在 L==R 的情况

✓ 循环条件 L <= R

用 L <= R，确保 L==R 时还能再判断一次

边界 case：
target < 所有元素：L=0 保持不变，返回 0
target > 所有元素：L 最终会变成 len(nums)，返回 len(nums)

核心

标准二分

L=0, R=n-1, while L <= R，每次缩半

关键

找不到返回 L

循环结束时 L 恰好是插入位置

变形

最左插入 / 最右插入

改 nums[mid] < target 判断，找边界
