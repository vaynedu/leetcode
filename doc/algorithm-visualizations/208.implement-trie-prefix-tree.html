LeetCode 208 · Trie

实现 Trie（前缀树）

高效存储和搜索字符串前缀的数据结构

核心洞察

Trie（发音 "try"）是一种树形结构，专门用于高效地存储和搜索字符串前缀。
核心思想：每个节点代表一个字符，从根到节点的路径代表一个前缀。

root (isEnd=F) └─ 'a' (isEnd=F) ├─ 'p' (isEnd=F) │ └─ 'p' [✓] (isEnd=T) │ └─ 'l' (isEnd=F) │ └─ 'e' [✓] (isEnd=T) └─ 'd' (isEnd=F) └─ 'o' (isEnd=F) └─ 'g' [✓] (isEnd=T)

存储了 "apple" 和 "dog"。节点标记 [✓] 表示是一个单词的结尾。

TrieNode 结构

每个节点包含：子节点映射（26个字母）+ 是否为单词结尾的标记

type TrieNode struct {
    children map[rune]*TrieNode  // 26个小写字母的子节点映射
    isEnd    bool               // 是否是某个单词的结尾
}

func NewTrieNode() *TrieNode {
    return &TrieNode{
        children: make(map[rune]*TrieNode),
        isEnd:    false,
    }
}

完整 Go 实现

type Trie struct {
    root *TrieNode
}

func Constructor() Trie {
    return Trie{root: NewTrieNode()}
}

func (t *Trie) Insert(word string) {
    node := t.root
    for _, ch := range word {
        if _, ok := node.children[ch]; !ok {
            node.children[ch] = NewTrieNode()
        }
        node = node.children[ch]
    }
    node.isEnd = true
}

func (t *Trie) Search(word string) bool {
    node := t.searchPrefix(word)
    return node != nil && node.isEnd
}

func (t *Trie) StartsWith(prefix string) bool {
    return t.searchPrefix(prefix) != nil
}

func (t *Trie) searchPrefix(prefix string) *TrieNode {
    node := t.root
    for _, ch := range prefix {
        if _, ok := node.children[ch]; !ok {
            return nil
        }
        node = node.children[ch]
    }
    return node
}

复杂度分析

时间复杂度

O(m)

m = 字符串长度，所有操作都只需遍历字符串长度

空间复杂度

O(m × n)

m = 平均字符串长度，n = 字符串数量

本质

前缀树

每个节点代表一个字符，从根到节点是前缀

核心

isEnd 标记

标记哪些节点是单词结尾，支持前缀搜索

优势

O(m) 时间

搜索/插入与字典大小无关，只与字符串长度相关
