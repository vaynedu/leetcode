Has Path Sum — 根到叶，值递减判断

问题定义

给定二叉树和目标值，判断是否存在从根到叶子的路径，使路径上所有节点值之和等于目标值。

目标22：5→4→11→2 = 22 ✓

递归核心

从根开始，目标值减去当前节点值，递归到左右子树。叶子节点时判断。

// 关键洞察：叶子节点才判断，非叶子继续递归 func HasPathSum(root *TreeNode, target int) bool { if root == nil { return false } // 叶子节点：判断是否等于目标值 if root.Left == nil && root.Right == nil { return root.Val == target } // 非叶子：减去当前值，递归左右子树 return HasPathSum(root.Left, target-root.Val) || HasPathSum(root.Right, target-root.Val) }

Trace：target=22

5→4→11→25不是叶子，target=22-5=17，递归左子树4
4→114不是叶子，target=17-4=13，递归左子树11
11→211不是叶子，target=13-11=2，递归右子树2
2 (叶子!)2是叶子且Val==target==2 → 返回true

模板口诀

核心思想

目标值递减传递，叶子节点判断是否归零

终止条件

nil返回false，叶子且值=目标返回true

递归式

左右子树或运算，任一找到即可
