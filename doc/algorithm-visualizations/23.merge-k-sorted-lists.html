复杂度分析

指标 值
时间 O(n log k)
空间 O(k)
n = 总节点数 k = 链表数量

为什么是 O(n log k)？

n 个节点，每个节点最多进堆出堆各一次。
每次堆操作 O(log k)，k 是链表数量（不是 n）。
