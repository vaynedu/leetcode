LeetCode 648 · Trie

单词替换

在句子中，将每个单词替换为词根前缀中最短的那个

核心洞察

题目：给定一个词根字典和一个句子，将句子中的每个单词替换为词根字典中以该单词开头且最短的前缀。
关键：用 Trie 存储词根，对每个单词从前缀树中找最短匹配词根。

root
├─ 'c'
│  └─ 'a'
      └─ 't'
         └─ isEnd=true ✓ cat
├─ 'b'
│  └─ 'a'
         └─ 't'
            └─ isEnd=true ✓ bat
└─ 'r'
    └─ 'a'
        └─ 't'
           └─ isEnd=true ✓ rat

词根: ["cat","bat","rat"]，每个节点标记 isEnd=true 的是词根

手推全过程

词根: ["cat","bat","rat"]，句子: "the cattle was rattled by the baby"

Step 1 处理 "the" → 在 Trie 中找不到前缀，保持原样
t→h→e：'t' 不在 root 子节点中 → 不替换
the
Step 2 处理 "cattle" → c→a→t→t→l→e，catt 是词根 "cat" 的前缀
c→a→t：节点 isEnd=true，找到词根 "cat"
cattle → cat
Step 3 处理 "was" → w 不在 Trie 中，不替换
Step 4 处理 "rattled" → r→a→t，rat 是词根
rattled → rat
结果 "the cat was rat by the baby"

Go 代码模板

func replaceWords(dictionary []string, sentence string) string {
    // 1. 构建 Trie
    root := NewTrieNode()
    for _, word := range dictionary {
        node := root
        for _, ch := range word {
            if _, ok := node.children[ch]; !ok {
                node.children[ch] = NewTrieNode()
            }
            node = node.children[ch]
        }
        node.isEnd = true
    }

    // 2. 处理每个单词
    words := strings.Split(sentence, " ")
    for i, word := range words {
        words[i] = findRoot(word, root)
    }
    return strings.Join(words, " ")
}

func findRoot(word string, root *TrieNode) string {
    node := root
    var prefix []byte
    for _, ch := range []byte(word) {
        if _, ok := node.children[ch]; !ok {
            break
        }
        node = node.children[ch]
        prefix = append(prefix, ch)
        if node.isEnd {
            return string(prefix)  // 找到最短词根
        }
    }
    return word  // 没找到，返回原词
}

复杂度分析

时间复杂度

O(m × n)

m=平均词根长度，n=句子单词数

空间复杂度

O(m × k)

m=词根数，k=平均词根长度

核心

Trie 找最短前缀

沿 Trie 走，遇到 isEnd=true 就停止

关键

最短词根

遇到第一个 isEnd=true 的节点就返回
