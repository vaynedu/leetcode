用队列实现栈

用一个或两个队列实现 LIFO 栈的 push、pop、top 操作

两种解法对比

方案 A：单队列

push 时将前面 n-1 个元素出队再入队，将新元素转到队首
push O(n)，pop O(1)

方案 B：双队列（推荐）

q1 持有栈元素，q2 作为中转
push O(1)，pop 均摊 O(1)

q1（主力）

—

front → rear

q2（中转）

—

front → rear

手推全过程（双队列）

依次执行：push(1), push(2), push(3), pop(), pop()

Op 1~3 push(1), push(2), push(3) → q1 直接入队

q1

1
2
3

top = 3（栈顶）

q2

空
Op 4 pop() → 3
q1: [1, 2, 3]，将前 n-1 个（1, 2）倒入 q2
q1 剩下 3，直接 pop/remove 即可

q1（结果）

1
2

top = 2

q2（备用）

空
栈顶 3 弹出，现在栈顶 = 2
Op 5 pop() → 2
q1: [1, 2]，将 1 倒入 q2
q1 剩下 2，弹出
栈顶 = 1

Go 代码模板（双队列）

type MyStack struct {
    q1 []int  // 主队列
    q2 []int  // 中转队列
}

func Constructor() MyStack {
    return MyStack{q1: []int{}, q2: []int{}}
}

// push: O(1) — 直接入 q1
func (s *MyStack) Push(x int) {
    s.q1 = append(s.q1, x)
}

// pop: 均摊 O(1)
func (s *MyStack) Pop() int {
    // 将 q1 前 n-1 个倒入 q2
    for len(s.q1) > 1 {
        s.q2 = append(s.q2, s.q1[0])
        s.q1 = s.q1[1:]
    }
    top := s.q1[0]
    s.q1 = s.q1[1:]
    // 交换 q1 q2 引用
    s.q1, s.q2 = s.q2, s.q1
    return top
}

func (s *MyStack) Top() int {
    // top = Pop() 后再 Push 回去
    val := s.Pop()
    s.q1 = append(s.q1, val)
    return val
}

func (s *MyStack) Empty() bool {
    return len(s.q1) == 0
}

单队列变种（push O(n)）

单队列每次 push 时，将前面 n-1 个元素出队再入队，把新元素转到队首。

push(4) 到 [1,2,3]：
出队入队 1, 2, 3 → q1: [2,3,1]
出队入队 2, 3 → q1: [3,1,2]
出队入队 3 → q1: [1,2,3,4]
完成！4 在队首（栈顶）
// 单队列 push: O(n)
func (s *MyStack) Push(x int) {
    s.q1 = append(s.q1, x)
    // 将前面 n-1 个元素循环移动到末尾
    for i := 0; i < len(s.q1)-1; i++ {
        s.q1 = append(s.q1, s.q1[0])
        s.q1 = s.q1[1:]
    }
}

核心

q1/q2 互换

pop 时把 n-1 个倒入 q2，最后一个弹出，然后交换引用

时间

push O(1)，pop 均摊 O(1)

比单队列方案优秀，推荐双队列
