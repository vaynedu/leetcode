二叉树的最近公共祖先

LCA — 后序遍历，左右子树分别找到 p 和 q

问题定义

给定二叉树和两个节点 p、q，返回它们的最近公共祖先（两个节点的祖先中深度最大的）。

p=节点2，q=节点5，LCA=3

后序递归核心

func LowestCommonAncestor(root, p, q *TreeNode) *TreeNode { // 基准情况：空节点 或 找到 p 或 找到 q if root == nil || root == p || root == q { return root } // 后序遍历：在左右子树中查找 left := LowestCommonAncestor(root.Left, p, q) right := LowestCommonAncestor(root.Right, p, q) // 左右都找到了 → 当前是 LCA if left != nil && right != nil { return root } // 只有一边找到 → 返回那边 if left != nil { return left } return right }

Trace：找节点 5 和 2

遍历 3后序查左右子树
查 5 的左递归到 5，5==p，返回 5
查 5 的右递归到 2，2==q，返回 2
5 左右都找到了返回 5 → 5 是 LCA!

模板口诀

基准返回

空节点/找到p/找到q → 直接返回

后序返回

左右子树分别查找，返回值决定身份

LCA判定

左右都非nil → 当前是LCA；否则返回非nil的那个
