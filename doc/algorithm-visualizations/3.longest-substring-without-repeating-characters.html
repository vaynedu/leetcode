无重复最长子串

变长窗口 + hashmap 记录位置，遇见重复就收缩

核心洞察

题目：给定字符串 s，找不含重复字符的最长子串长度。
关键：右指针扩展，遇到重复字符时，左指针直接跳到重复字符的右侧。

Step 1: "abc" → 遇到 b 重复

关键洞察

当 R 遇到重复字符 b：
→ map 记录 b 上次出现在 index 2
→ L = max(L, pos+1) = max(0, 3) = 3
→ 窗口从 abc 收缩成 cb

为什么 L 只能用 max 前进？
因为 L 之前可能已经被拉到这个重复字符之后了（比如遇到多个重复字符逐个收缩）。 用 max 保证 L 永远不会被拉回窗口外。

Go 代码模板

变长窗口，hashmap 记录每个字符最新出现的下标

func lengthOfLongestSubstring(s string) int {
    last := make(map[byte]int)  // 记录字符上次出现的下标
    L := 0
    maxLen := 0

    for R := 0; R < len(s); R++ {
        if pos, exists := last[s[R]]; exists && pos >= L {
            L = pos + 1  // L 跳到重复字符的右侧
        }
        last[s[R]] = R
        maxLen = max(maxLen, R-L+1)
    }
    return maxLen
}

易错点

L 只能用 max 前进，不能直接赋值

❌ 错误写法

if pos, exists := last[s[R]]; exists {
    L = pos + 1  // 危险！可能把 L 拉回窗口外
}

✓ 正确写法

if pos, exists := last[s[R]]; exists && pos >= L {
    L = max(L, pos+1)
}

场景：s = "abba"
当 R=3 遇到第二个 b 时，pos=1，如果直接 L=pos+1=2（正确），但如果之前 L 已经被拉到 3 了， 直接赋值会把 L 拉回 2。用 max 永远保证 L 只进不退。
