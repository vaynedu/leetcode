无重复最长子串

右指针扩展窗口，遇到窗口内重复字符时，左指针直接跳到重复字符右侧。

核心洞察

题目：给定字符串 s，找不含重复字符的最长子串长度。
关键：窗口内永远保持无重复；一旦 s[R] 在窗口内出现过，L 只向右跳，不回退。

观察

重复只影响当前窗口

如果字符上次出现位置在 L 左侧，它已经不在窗口里，对当前窗口没有影响。

滑动选择

L 直接跳过旧字符

遇到窗口内重复字符时，令 L = max(L, last[ch]+1)，一次跳到合法窗口。

复杂度

O(n) 时间 O(k) 空间

每个字符只被右指针扫描一次，hashmap 记录字符最近一次出现位置。

Go 代码

变长窗口，hashmap 记录每个字符最新出现的下标。

func lengthOfLongestSubstring(s string) int {
    last := make(map[byte]int)
    left := 0
    ans := 0

    for right := 0; right < len(s); right++ {
        ch := s[right]
        if pos, ok := last[ch]; ok && pos >= left {
            left = pos + 1
        }
        last[ch] = right
        ans = max(ans, right-left+1)
    }
    return ans
}

易错点

左指针只能向右走，不能被旧重复字符拉回去。

错误：只判断是否出现过

if pos, ok := last[ch]; ok {
    left = pos + 1
}

s = "abba"，处理最后一个 a 时，旧 a 在 0，但当前 left 已经是 2，直接赋值会把 left 拉回 1。

正确：确认旧字符在当前窗口内

if pos, ok := last[ch]; ok && pos >= left {
    left = pos + 1
}

只有 pos >= left 时才说明重复发生在当前窗口内，left 才需要前进。

核心

变长窗口

右指针负责扩展，左指针只在重复字符落在当前窗口内时前进。

数据结构

hashmap 记录位置

key 是字符，value 是最近一次下标，用来 O(1) 判断重复是否在窗口内。

关键细节

left 只能前进

判断 pos >= left，避免旧重复字符把左边界拉回窗口外。
