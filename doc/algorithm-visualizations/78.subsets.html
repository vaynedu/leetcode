给定无重复数组，返回所有可能的子集

核心洞察

题目：给定一个无重复数组，返回所有可能的子集（幂集）。
关键：每个元素都有"选"和"不选"两种状态，共 2ⁿ 个子集。本质是一棵二叉树的遍历。

每个元素：选 / 不选 → 2³ = 8 个子集

二叉树视角

每层代表一个元素，分叉为"选"或"不选"

                []          depth=0（空集）
           /         \
        []            [1]      depth=1（选/不选元素1）
       /   \         /   \
    []      [2]   [1]   [1,2]  depth=2
    / \     /\    / \    / \
  []  [3] [2][2,3][1][1,3][1,2][1,2,3]  depth=3

共 2³ = 8 个子集

手推全过程

nums = [1,2,3]

初始 path=[], index=0
depth=0 不选 1 → [], 选 1 → [1]
[] [1]
depth=1 在 [] 基础上：不选2→[], 选2→[2]
[] [1] [] [2]
depth=2 在 [1] 基础上：不选2→[1], 选2→[1,2]
[] [1] [] [2] [1] [1,2]
完成 共 8 个子集
[] [1] [2] [3] [1,2] [1,3] [2,3] [1,2,3]

Go 代码模板

func subsets(nums []int) [][]int {
    var result [][]int
    var path []int

    var backtrack func(index int)
    backtrack = func(index int) {
        // 每个节点都是合法子集，直接收集
        ans := make([]int, len(path))
        copy(ans, path)
        result = append(result, ans)

        for i := index; i < len(nums); i++ {
            path = append(path, nums[i])
            backtrack(i + 1)  // 关键：从 i+1 开始（不回头选）
            path = path[:len(path)-1]
        }
    }

    backtrack(0)
    return result
}
