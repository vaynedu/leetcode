32. 最长有效括号

Longest Valid Parentheses — 栈底记录"最后一个未匹配的右括号"

核心思想

栈底永远保存「最后一个未匹配的右括号下标」。遇到左括号 ( 入栈，遇到右括号 ) 出栈。
有效括号长度 = 当前下标 − 栈顶下标。

输入示例

s = )()())
答案 = 4
)0 (1 )2 (3 )4 )5

结果

maxLen = 4
有效括号段：索引 1–4 ()( )

核心规律

为什么需要「哨兵」？

栈初始化时放入 -1 作为哨兵。计算长度时：
len = i − stack.top()
当所有括号都匹配完，栈顶回到哨兵位置，长度自然归零，不会出错。

时间空间

时间 O(n) — 每个字符最多入栈出栈各一次
空间 O(n) — 最坏情况全是左括号
