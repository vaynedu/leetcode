LeetCode 159 · Sliding Window

至多包含两个不同字符的最长子串

滑动窗口变种：维护窗口内最多两种字符

核心洞察

题目：给定一个字符串，找出至多包含两种不同字符的最长子串长度。
这是 #3（无重复字符）的变种，#3 用 hash set 判断"无重复"，本题用 hash map 统计每种字符的出现次数。

核心思路：

维护一个 sliding window [left, right)，用 hash map 记录窗口内每个字符的出现次数。

当出现第 3 种不同字符时，收缩左边界直到窗口内只有 2 种字符。

#3 无重复字符

hash set 记录已见字符
遇到重复 → 收缩左边界直到移除

#159 最多2种字符

hash map 记录每种字符的 count
种类>2 → 收缩左边界直到移除一种

最终答案：maxLen = 3（子串 "ece" 或 "eab"）

Go 代码模板

func lengthOfLongestSubstringTwoDistinct(s string) int {
    hash := make(map[byte]int)
    left := 0
    maxLen := 0

    for right := 0; right < len(s); right++ {
        // 扩大窗口：加入 right 位置的字符
        hash[s[right]]++

        // 收缩窗口：保证窗口内最多2种字符
        for len(hash) > 2 {
            hash[s[left]]--
            if hash[s[left]] == 0 {
                delete(hash, s[left])
            }
            left++
        }

        // 更新最大长度
        maxLen = max(maxLen, right-left+1)
    }
    return maxLen
}

与 #3 的关键区别

#3 无重复字符

hash set 记录已见字符
遇到重复就收缩
只需要记录"是否出现过"

#159 最多2种字符

hash map 记录 count
种类超过2就收缩
需要记录"每个字符出现几次"

核心

hash map 计数

记录窗口内每种字符的出现次数

关键

len(hash) > 2

种类超过2时收缩左边界

变种

扩展到 k 种字符
