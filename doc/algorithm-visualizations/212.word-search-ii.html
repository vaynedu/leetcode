单词搜索 II

在二维网格中找到所有出现在网格中的单词

核心洞察

题目：给定一个 m×n 的字符网格和一个单词列表，找到所有可以由网格中的相邻单元格字母连接形成的单词。
关键：先构建 Trie，然后从每个格子开始 DFS 回溯搜索。

搜索过程

从 't' 开始（e,t 单元格）搜索，"oath" 从左上角开始搜索

Step 1 从 't' 开始，上下左右 DFS
t → te → tea → tean（不对）
t → to → too → toom（不对）
t → oa → oat → oath ✓
Step 2 回溯尝试其他路径
从 'e' 开始 (1,1)
e → ea → eat ✓
完成 找到: oath, eat, teen
oath eat teen

Go 代码模板

func findWords(board [][]byte, words []string) []string {
    // 1. 构建 Trie
    trie := Constructor()
    for _, w := range words {
        trie.Insert(w)
    }
    m, n := len(board), len(board[0])
    visited := make([][]bool, m)
    for i := range visited {
        visited[i] = make([]bool, n)
    }
    var result []string

    // 2. 从每个格子开始 DFS
    var dfs func(i, j int, node *TrieNode, path []byte)
    dfs = func(i, j int, node *TrieNode, path []byte) {
        if i < 0 || i >= m || j < 0 || j >= n || visited[i][j] {
            return
        }
        ch := board[i][j]
        if child, ok := node.children[ch]; ok {
            path = append(path, ch)
            visited[i][j] = true
            if child.isEnd {
                result = append(result, string(path))
                child.isEnd = false  // 避免重复
            }
            // 继续 DFS
            dirs := [][2]int{{-1,0},{1,0},{0,-1},{0,1}}
            for _, d := range dirs {
                dfs(i+d[0], j+d[1], child, path)
            }
            visited[i][j] = false
            path = path[:len(path)-1]
        }
    }

    for i := 0; i < m; i++ {
        for j := 0; j < n; j++ {
            dfs(i, j, trie.root, nil)
        }
    }
    return result
}

重要优化：剪枝

不加优化

找到单词后不标记，相同单词会被重复加入结果

优化：标记已找到

child.isEnd = false 防止重复加入
