435. 无重叠区间

贪心：移除最少区间使不重叠 → 按右边界排序，贪心保留结尾最早的

阶段: -
当前区间: -
已移除: 0
已保留: 0

结果
function eraseOverlapIntervals(intervals) {
  intervals.sort((a,b) => a[1] - b[1]);
  let count = 0, end = intervals[0][1];
  for (let i = 1; i < intervals.length; i++) {
    if (intervals[i][0] < end)
      count++;
    else
      end = intervals[i][1];
  }
  return count;
}
