Step 1 · 切

先切一刀

取 mid = L + (R-L)/2。若 nums[mid] == target，直接返回。

Step 2 · 判

判断哪半有序

nums[L] ≤ nums[M] 则左半有序，否则右半有序。必有一半有序。

Step 3 · 夹

用有序半夹 target

看 target 在不在有序那半的首尾之间：在 → 去那半；不在 → 去另一半。

Time · O(log n)  ·  Space · O(1)  ·  Only one pivot per array
